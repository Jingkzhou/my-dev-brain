# 不良贷款-报表口径

## 概念定义

**不良贷款**：贷款五级分类中次级类、可疑类、损失类之和。

依据：《贷款风险分类指引》《商业银行金融资产风险分类办法》（2023年第1号令）。

### 五级分类定义（全报表通用）

| 等级      | 归类     | 认定标准                        |     |
| ------- | ------ | --------------------------- | --- |
| **正常类** | 良好     | 债务人能够履行合同，无客观证据表明不能按时足额偿付   |     |
| **关注类** | 良好     | 存在不利因素，但债务人目前有能力偿付          |     |
| **次级类** | **不良** | 债务人无法足额偿付本金/利息，或金融资产已发生信用减值 |     |
| **可疑类** | **不良** | 已无法足额偿付，金融资产已发生**显著**信用减值   |     |
| **损失类** | **不良** | 采取所有措施后，只能收回极少部分或全部损失       |     |

**不良贷款 = 次级 + 可疑 + 损失**

## 逾期贷款口径

### 通用定义
逾期贷款：报告期末，未按合同规定期限偿还本金或利息的贷款**本金余额**。

### 常见逾期分档

| 档次 | 适用报表 |
|------|----------|
| 逾期60天以上 | G01_II、S64、S72 |
| 逾期90天以上 | G01_II、S67、S72 |
| 逾期1-60/61-90/91天以上三档 | S67 |
| 逾期31-60/61-90/91-180/181-270/271-360/361+天 | S63、S71 |

### 特殊逾期规则

**个人住房贷款（S67）**：
- 逾期 < 90天：仅填**已逾期部分**本金余额
- 逾期 ≥ 90天：填**整笔贷款**本金余额

**按月分期个人消费贷款（G01_II）**：
- 逾期 ≤ 90天：仅填已逾期本金余额
- 逾期 91天以上：填整笔贷款本金余额

**借新还旧**：逾期天数从**原合同**到期日起算

**正常展期**：逾期天数从**展期后**到期日起算

## 易混淆点

1. **逾期贷款 vs. 不良贷款**
   - 不是等价关系：逾期未必不良，不良未必逾期
   - G01_II 同时报两个维度，不要混用
   - 通常要求：逾期90天以上必须认定为不良（但两者口径独立统计）

2. **各专项表的不良口径**
   - S68（绿色）、S70（科技）、S71（普惠）、S72（制造业）等专项表各自按专项口径的贷款范围计算不良，**不代表全行口径**
   - 汇总检验时，专项表不良不等于全行不良

3. **G12 的处置核对**
   - 多张专项表都要对齐 G12，但 G12 目前**尚未入库**，遇到此类核对关系需待确认

4. **不良率口径**
   - G53 中不良率 = 不良贷款 / 各项贷款（按地区发放口径）
   - 各专项表中分母仅为该专项范围贷款，与全行不良率不同
