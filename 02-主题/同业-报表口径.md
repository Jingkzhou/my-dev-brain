# 同业-报表口径

## 概念定义

**同业**：填报机构以外的其他银行业金融机构及非银行金融机构。

1104 系统中同业业务涉及两个维度：
- **资产负债分类**：同业往来在资产负债表中如何归类（G01 系列）
- **风险暴露计量**：同业客户的大额风险暴露如何报告（G14 系列）
- **流动性管理**：同业资产/负债的期限分布（G21）
- **利率风险**：同业工具对银行账簿利率敏感性的影响（G33）
- **存款口径**：地区分类等报表中同业存款是否纳入（G53 等）

## 同业业务分类口径

### 核心业务类型（各表通用定义，来源：银发〔2014〕127号）

| 业务类型 | 定义要点 |
|----------|----------|
| **同业拆借** | 经人民银行批准，通过全国统一市场拆借，无担保 |
| **同业借款** | 经批准的金融机构间资金借贷 |
| **同业存款** | 向有吸收存款资质的机构存入款项 |
| **同业代付** | 受托行代委托行对外付款，受托行形成对委托行拆放 |
| **买入返售** | 以协议方式买入金融资产并约定到期返售，抵押物所有权不转移 |
| **同业存单** | 在全国银行间债券市场发行/持有的记账式存款凭证 |
| **同业债券** | 银行账簿中持有的金融机构发行债券（金融债、可转债、次级债等，不含 ABS 及交易账簿债券） |

### 政策性金融债的特殊归属
属于同业债券子类，发行主体限于：国家开发银行、中国进出口银行、中国农业发展银行（在银行间市场、沪深交易所或商业银行柜台发行）。

## 易混淆点

1. **结算性同业存款 vs. 投融资性同业存款**
   - 结算性：仅用于结算用途（银发〔2014〕127号、178号），不含投融资性结算账户存款
   - 两者在部分报表口径下有区分，填报前需判断业务性质

2. **同业代付的归属**
   - 委托行：代付款项→[7. 贸易融资]（不归拆放同业）
   - 受托行：代付款项→[10. 拆放同业]（对委托行的债权）

3. **农信系统内部资金**
   - 农信社与省联社或其他农信社之间的系统内资金调剂，按同业拆放/存放/买入返售/卖出回购处理，不视为内部往来

4. **保本理财的同业归属**
   - 买入其他机构发行的保本理财→存放同业款项（不归投资）
   - 本行发行的保本理财被他行持有→同业存放款项

5. **G14_V 与 G14_VI 的门槛差异**
   - G14_V（单一）：无论是否超 2.5% 门槛，固定填前 100 名（压力测试要求）
   - G14_VI（集团）：超过 2.5% 才报，最多 30 名

6. **G01_Ⅶ 金融业贷款排除同业**
   - G01_Ⅶ 中贷款投向行业分类的「金融业」，明确不包括对同业的拆放、贴现、转贴现
   - 即同业拆放不计入金融业贷款，两类统计口径互斥

7. **G53 存款不含同业**
   - 分地区存款口径（G53）仅含单位存款+个人存款，同业存款单独处理，不纳入地区统计存款规模
