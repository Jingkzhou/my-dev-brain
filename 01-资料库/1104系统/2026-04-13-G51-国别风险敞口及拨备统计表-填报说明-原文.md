## 第一部分：引言

本报表旨在收集银行业金融机构国别风险债权、债务敞口及相关拨备情况，以此作为评价银行业金融机构国别风险的依据之一。本报表根据《银行业金融机构国别风险管理办法》制定。

## 第二部分：一般说明

1. 报表名称：国别风险敞口及拨备统计表
2. 报表编码：银监统（空缺）
3. 填报机构：政策性银行（开发银行）、大型商业银行（含邮政储蓄银行）、股份制商业银行、城市商业银行、外资法人银行、外国银行分行
4. 报送频度：并表口径，季后40日内。出现重大事件可能对国别风险产生实质性不利影响时，按监管要求及时报送。

7. 四舍五入要求：金额保留两位小数。
8. 填报币种：本表要求以本外币合计人民币数据填报。银行业金融机构将外币折算为人民币时，应按照报告期末最后一天国家外汇管理局公布的人民币兑美元、欧元、日元和港币的基准汇价进行折算。美元、欧元、日元和港币四种主要货币以外的其他货币对人民币的折算汇率，以报告期末最后一天美元对人民币的基准汇率与同一天上午9时国际外汇市场其他货币兑美元汇率套算确定。

## 第三部分：具体说明

### 总体说明：境外主体定义

本表中各项境外债权债务敞口按照国家分别填报。国家的划分以联合国宪章和我国外交相关法律法规及惯例为依据。

**"境外"的定义**：除中华人民共和国以外的其他国家或地区，以及中国香港、澳门和台湾。

**境外主体**：包括国家、营业场所在境外的金融机构和非金融企业、以及居住在境外的个人等，含境外银行在境内的分行。

**境外个人**：居住在国外的个人。多国有住所的，按经常居住地确定国别。

**境外企业（含金融机构）的判定三条原则**：
1. 依据外国（地区）法律成立；
2. 实际管理机构不在中国境内；
3. 主要经营活动在境外。

如果依据一国法律成立，但主要经营活动在另一国，按主要经营活动所在国确定国别。在离岸中心注册但主要经营活动不在该离岸中心的境外企业，债权填列在其主要经营活动所在国。

**境外分公司（不能独立承担法律责任的分公司）视为境外主体**，直接境外债权填列在其总公司所在国。

**填报机构对分行的处理规则**：
- 对中资商业银行的境外分行和外资银行的境外分行的直接境外债权：填列在分行所在国，并按风险转移规定体现为对其母行的最终境外债权
- 对外资银行的境内分行的直接境外债权：直接填列在母行所在国，且不作风险转移

### 境外债权的范围

**纳入"境外债权"的项目**（11类）：
1. 存放同业
2. 存放境外中央银行
3. 境外有价证券和基金投资
4. 境外贷款（含银团贷款）
5. 境外金融衍生品价值（处于盈利的合约正总市场价值）
6. 对非并表境外机构的投资（股权投资总额）
7. 买入返售资产
8. 拆放同业
9. 境外票据融资
10. 境外应收账款
11. 其他境外债权

**不纳入"境外债权"的项目**：
1. 境外房产
2. 其他境外不动产
3. 境外机构的商誉

### 行项目详细说明

**【1. 直接境外债权】**：填报机构持有的境外债务人发行的有价证券或其他相关债权。
  - **【1.1 跨境债权】**：填报机构持有的跨境债权，含境内机构与境外主体的债权，以及境外机构与除中国及该境外机构所在国以外的其他国家客户的债权，不含【1.2】。
  - **【1.2 境外分支机构债权】**：填报机构的境外机构对当地客户发生的债权。
    - **【1.2.1 当地货币债权】**：以当地法定货币计价的债权。
    - **【1.2.2 非当地货币债权】**：以非当地法定货币计价的债权。

各债权均可按以下三个维度进一步分解：
- **债务人类型**（6类）：银行 / 公共部门 / 非银行金融机构 / 非金融机构 / 个人 / 其他
- **融资工具类型**（7类）：境外贷款（含银团贷款）/ 存放同业+拆放同业+买入返售 / 存放中央银行 / 境外有价证券和基金投资 / 境外金融衍生品 / 对非并表境外机构的投资 / 其他境外债权
- **剩余期限**（4档）：1年以下（含1年）/ 1-2年（含）/ 2年以上 / 其他（无固定到期日或期限不可知）

**债务人类型定义**：
- **银行**：获注册地监管当局认可的银行业金融机构，含商业银行和其他存款金融机构，但不包括该国中央银行。
- **公共部门**：外国政府和官方机构，包括：①中央和地方政府及其部门和机构；②政府拥有的履行公共职能的银行（如政策性银行）；③国际或区域组织（IMF、WB、BIS、ADB等）。
- **非银行金融机构**：《国民账户体系》金融公司部门中，除央行及其他存款类金融机构以外的金融机构，主要包括贷款公司、信托投资公司、财务公司、汽车金融服务公司、消费金融公司、证券公司、保险公司、金融辅助机构、养老基金、金融中介机构及其他。
- **非金融机构**：具有独立法人资格，未包含在银行、公共部门和非银行金融机构中的企事业单位。
- **个人**：具有独立民事行为能力的城乡居民和个体生产者、经营者。
- **其他**：上述五类以外的所有企业、个人和其他组织。

**【2. 风险转移】**：境外债务人通过风险转移手段将债权风险转移的净额（转入-转出）。

风险转移手段包括：
1. 第三方提供的有法律效力的担保（含备用信用证），但不包括填报机构总行及其并表机构提供的担保
2. 保险（借款人违约或货币无法转换时由保险人支付）
3. 境外债务人总行提供的担保（境外银行分行的债权视同该行总行提供担保）
4. 信用衍生品（购买信用衍生合约提供的担保）
5. 合格抵质押品和其他风险缓释工具（仅缓释信用风险但不转移国别风险的，不列入）

规则：转入方国别风险评级通常须优于转出方。如发生净转入填正数，净转出填负数，同时在转入/转出国计入同等金额的正/负数。

- **【2.1 跨境债权风险转移】** = 2.1.1（风险转入）- 2.1.2（风险转出）
- **【2.2 境外分支机构债权风险转移】** = 2.2.1（风险转入）- 2.2.2（风险转出）

**【3. 最终境外债权】** = 【1. 直接境外债权】+ 【2. 风险转移】

由中国大陆主体提供风险转移且全额覆盖直接境外债权的，在最终债权中不再反映；部分覆盖的，覆盖部分不再反映。

- **【3.1 跨境债权】**（含债务人类型和融资工具类型分解）
- **【3.2 境外分支机构债权】**（含债务人类型和融资工具类型分解）

**【4. 承诺和担保及金融衍生品】** = 4.1 + 4.2 + 4.3
- **【4.1 未提取承诺】**：对境外主体的未提取承诺
- **【4.2 担保】**：对境外主体的担保（总行及境内外分支机构）
- **【4.3 金融衍生品】**：对境外主体的金融衍生合约（按名义价值）
  - **【4.3.1 其中：市值为正的金融衍生品名义价值】**

**【5. 国别风险限额】**：按《银行业金融机构国别风险管理办法》要求设置的国别风险限额。

**【6. 资本净额】**：与1104系统 G40 报表的【3.A】资本净额保持一致。外国银行分行按本外币合计口径填报。

**【7. 最终境外债权占资本净额的比重】** = 【3.】÷【6.A】

**【8. 境外债务】** = 8.1（跨境债务）+ 8.2（境外分支机构债务）

境外债务包括：吸收境外存款、与境外金融机构的同业存放、对境外金融机构的同业拆入、与境外主体的卖出回购、对境外主体的应付利息、对外国税务机关的应交税金、以及其他境外债务。

按类别划分（8.3）：
- **【8.3.1 存款】**：从境外客户或境外分支机构吸收的各项存款，含不可转让存单
- **【8.3.2 同业融入】**：从境外同业机构融入的资金，不包括集团内的
- **【8.3.3 应付债券】**：向境外客户或境外分支机构发行的债务工具本金及应付利息，含可转让存单
- **【8.3.4 衍生品负债】**：与境外交易对手或境外分支机构开展衍生品业务形成的衍生负债公允价值
- **【8.3.5 其他】**：其他来自境外客户或境外分支机构的负债

**【9. 应计提拨备的国别风险资产】**：考虑风险转移和风险缓释后的国别风险资产，含境外债权及担保承诺等表外业务。

**【10. 最终境外债权减值准备】**：按财政部《金融企业准备金计提管理办法》已计提的减值准备。

**【11. 国家分类】**：按《银行业金融机构国别风险管理办法》对国家/地区所做的分类：
低=1 / 较低=2 / 中等=3 / 较高=4 / 高=5 / 无=0

**【12. 应计提国别风险拨备比率】**：各等级最低拨备比率：
- 低国别风险、较低国别风险：0%
- 中等国别风险：不低于5%
