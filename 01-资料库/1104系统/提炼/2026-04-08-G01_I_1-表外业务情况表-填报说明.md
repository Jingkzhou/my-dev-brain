## 表一：表外业务分类（四大类）

| 类别 | 说明 | 关键项目 |
|------|------|----------|
| **担保承诺类** | 承担偿付责任或提供信用服务 | 承兑汇票、跟单信用证、保函、销售与购买协议、贷款承诺 |
| **代理投融资服务类** | 不承担代偿责任、不承诺投资回报 | 发行理财产品、委托贷款、委托投资、代理交易、代理代销 |
| **中介服务类** | 提供中介服务、收取手续费 | 资产托管、代理收付、保管业务、财务顾问咨询 |
| **其他类** | 除上述外的其他表外业务 | 金融衍生品、其他中介服务 |

## 重要口径说明

### 委托贷款（[14.]）

| 特点 | 说明 |
|------|------|
| 资金来源 | 委托人提供资金，填报机构**不得代垫资金** |
| 风险承担 | 风险由委托人承担 |
| 分类 | [14.1]现金管理项下、[14.2]金融机构委托、[14.3]非金融机构委托 |
| 公积金委托贷款 | 专指住房公积金管理中心委托发放的住房贷款 |

### 委托投资（[15.]）

| 特点 | 说明 |
|------|------|
| 资金来源 | 委托人提供资金，填报机构**不得代垫资金** |
| 风险承担 | 风险由委托人承担 |
| 排除范围 | **不含**填报机构发行的表外理财产品 |

### 代理代销业务（[20.]）

| 细分项 | 说明 |
|--------|------|
| [20.1] | 代理代销信托计划（个人/法人合格投资者） |
| [20.2] | 代理代销资产管理计划（证券/保险/基金） |
| [20.3] | 代理代销保险产品（银保代理） |
| [20.4] | 代理代销基金（含基金专户，**不含**资产管理计划） |
| [20.5] | 代理代销银行理财产品 |
| [20.5.1] | 代理代销本行理财公司产品 |
| [20.5.2] | 代理代销养老理财产品 |
| [20.6] | 代理销售其他金融产品 |

### 金融衍生品类（[29.]）

| 特点 | 说明 |
|------|------|
| 业务类型 | 远期、期货、掉期（互换）、期权、结构化金融工具 |
| 填报方式 | **名义本金期末余额** |
| 排除范围 | **不包括计入**[16.1]代理金融衍生产品交易的衍生品 |
