## 核心指标及计算公式

| 编号 | 项目 | 计算公式 | 说明 |
|------|------|----------|------|
| [5] | 各项存款（调整后） | [1] + 境外母行一年期以上存放净额 | 仅外资法人银行需调整 |
| [6] | 月日均存款（调整后） | 每日调整后存款之和 / 当月天数 | |
| [7] | 各项贷款（调整后） | [3] - 6项扣减 | 扣减再贷款等6类贷款 |
| [8] | 月日均贷款（调整后） | 每日调整后贷款之和 / 当月天数 | |
| [9] | 调整后存贷比 | [7] / [5] | 调整后贷款/调整后存款 |
| [10] | 调整后月日均存贷比 | [8] / [6] | 调整后月日均贷款/调整后月日均存款 |
| [11] | 存款偏离度 | 按银保监办发[2018]48号计算 | 监管考核指标 |

---

## 调整项说明

### 存款调整 (+1项)

| 编号 | 调整项 | 说明 |
|------|--------|------|
| + | 外资法人银行吸收的境外母行一年期以上存放净额 | 仅外资法人银行适用 |

> **注意**：大额存单已直接纳入"各项存款"，此处不再重复加回。

### 贷款调整 (-6项)

| 编号 | 扣减项 | 说明 |
|------|--------|------|
| 1 | 支农再贷款、支小再贷款对应的贷款 | |
| 2 | "三农"专项金融债对应的涉农贷款 | |
| 3 | 小微企业专项金融债对应的小微企业贷款 | |
| 4 | 剩余期限>=1年且债权人无权提前偿付的债券对应的贷款 | |
| 5 | 使用国际金融组织或外国政府转贷资金发放的贷款 | |
| 6 | 村镇银行使用主发起行存放资金发放的农户和小微企业贷款 | |

## 核对关系

### 表内校验关系

| 校验式 | 说明 |
|--------|------|
| `[C] = [A] + [B]` | 适用于[1][2][3][4][5][6][7][8]（本外币合计 = 人民币 + 外币） |
| `[1.A] <= [5.A]` | 各项存款 <= 调整后存款（A列） |
| `[2.A] <= [6.A]` | 月日均各项存款 <= 调整后月日均存款（A列） |
| `[3.A] >= [7.A]` | 各项贷款 >= 调整后各项贷款（A列） |
| `[4.A] >= [8.A]` | 月日均各项贷款 >= 调整后月日均贷款（A列） |
| `[9.A] = [7.A] / [5.A]` | 调整后存贷比（A列） |
| `[10.A] = [8.A] / [6.A]` | 调整后月日均存贷比（A列） |
