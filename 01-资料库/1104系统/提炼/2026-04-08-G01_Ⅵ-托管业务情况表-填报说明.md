## 核心定义

### 托管业务定义

商业银行作为独立第三方，按照法律法规规定和托管合同约定，为所托管产品在中华人民共和国境内提供财产保管及相关服务的行为。

### 资金监管业务说明

| 性质 | 说明 |
|------|------|
| 是否托管 | 不属于托管业务 |
| 填报要求 | 在本表中列报（[13.]资金监管业务） |
| 业务类型 | 第三方资金存管、交易资金监管、受托支付监管 |

## 私募投资基金托管说明

| 细分项 | 说明 |
|--------|------|
| 统计口径 | 填报净资产余额，而非合同金额 |
| 排除范围 | 保险私募基金托管、QDLP/QDIE产品托管、QFLP产品托管 |
| 包含范围 | 私募证券、私募股权、其他私募（如资产配置类） |
| 基金结构 | 契约型、合伙型、公司型基金 |

### 保险资产托管说明

| 说明 | 说明 |
|------|------|
| 统计范围 | 保险集团/公司资金运用、保险资管产品 |
| 包含产品 | 组合类产品、保险债权计划、保险股权计划、保险私募基金、保险资产支持计划 |
| [7.1]统计 | 仅限银保监会指定机构备案的私募基金 |

### 养老金托管说明

| 细分项 | 统计口径 |
|--------|----------|
| [8.1]企业年金 | 已获取人社部门企业年金计划确认函的各类计划 |
| [8.2]职业年金 | 已获取人社部门职业年金计划确认函的职业年金计划 |
| [8.3]养老金产品 | 已向人社部产品备案的养老金产品 |

### QDII/QFI类产品说明

| 类别 | 说明 |
|------|------|
| QDII | 应包括QDII基金、QDII专户、QDII理财、QDII保险、QDII信托 |
| QFII | 包括QFII、RQFII、QFLP、境外机构直接投资银行间债券市场、GDR存托人、GDR跨境转换机构 |
