## 填报核心原则

| 原则 | 说明 |
|------|------|
| **核心原则** | 按**贷款资金实际用途**分类，而非借款人所属行业填报 |
| **具体规则** | 根据贷款合同中约定的资金用途进行填报 |
| **例外规则** | 无法合理确定贷款投向时，按借款人主营业务所属行业填报 |
| **个人贷款** | 个人经营性贷款也按此原则填报 |

## 关键口径说明

### 行业分类标准

| 项目 | 标准依据 |
|------|----------|
| [2.1-2.20] 行业分类 | GB/T4754-2017《国民经济行业分类》 |
| [4] 高技术产业 | 国统字[2017]200号《高技术产业分类》 |
| [5-6] 数字经济 | 国家统计局令第33号《数字经济及其核心产业统计分类（2021）》 |
| [7] 知识产权密集型产业 | 国家统计局令第25号《知识产权（专利）密集型产业统计分类（2019）》 |
| [8] 养老产业 | 国家统计局令第30号《养老产业统计分类（2020）》 |

### 特殊口径规则

| 项目 | 规则说明 |
|------|----------|
| [2.10] 金融业 | 不包括对同业的拆放、贴现和转贴现业务 |
| [2.21.3] 住房按揭贷款 | 不包括以个人住房作抵押但用途不为购房的贷款（商业用房、商住两用房） |
| [2.22] 买断式转贴现 | "买断"指无回购协议，转入方可自由支配票据 |
| [2.23] 买断其他票据类资产 | "买断"指无回购协议，**与是否有追索权无关** |
| [5.4]+[6] | 一笔贷款不应同时属于数字经济核心产业贷款和数字化效率提升业贷款 |
| [8] 养老产业 | 以每笔借据为标准，实际投向占比≥50%或借款人营业收入占比≥50% |

## 核对关系

### 表内校验

| 校验式 | 说明 |
|--------|------|
| `[1.] = [2.] + [3.]` | 各项贷款 = 对境内贷款 + 对境外贷款 |
| `[2.] = Σ(2.1-2.20) + [2.21] + [2.22] + [2.23]` | 对境内贷款 = 20行业 + 个人贷款 + 2项买断 |
| `[2.21] = [2.21.1] + [2.21.2] + [2.21.3] + [2.21.4]` | 个人贷款 = 4个细分项之和 |
| `[5] = [5.1] + [5.2] + [5.3] + [5.4]` | 数字经济核心产业 = 4个子类之和 |
| `[4] ≤ [2]` | 高技术产业 ≤ 对境内贷款（各列） |
| `[5] ≤ [2]` | 数字经济核心产业 ≤ 对境内贷款（各列） |
| `[6] ≤ [2]` | 数字化效率提升业 ≤ 对境内贷款（各列） |
| `[7] ≤ [2]` | 知识产权密集型产业 ≤ 对境内贷款（各列） |
| `[8] ≤ [2]` | 养老产业 ≤ 对境内贷款（各列） |

### 列校验

| 校验式 | 说明 |
|--------|------|
| `A ≥ B` | A列金额 ≥ B列金额（除[2.22][2.23]外） |
