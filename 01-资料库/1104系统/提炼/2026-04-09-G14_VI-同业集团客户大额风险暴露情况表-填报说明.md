## 主要内容结构

### 报表在 G14 体系中的位置

- G14 反映银行大额风险暴露总体情况以及各类客户风险暴露具体情况，共分六个部分。
- G14_VI 是第 VI 部分，聚焦“同业集团客户”维度。
- 本表与 G14 第 I、II、III、V 部分存在字段定义继承关系，不是完全独立口径。

### G14_VI 的客户筛选与排序逻辑

| 项目 | 规则 |
|------|------|
| 客户类型 | `[A]` 固定填写“同业集团客户” |
| 入表门槛 | 考虑合格风险缓释作用后风险暴露超过一级资本净额 2.5% |
| 填报范围 | 若客户不超过 30 个，全部填报；若超过 30 个，仅填前 30 家 |
| 排序方式 | 按 `[D]` 列降序排列 |

### 字段口径继承关系

| 字段 | 口径来源 |

## 值得单独沉淀的规则线索

### 关键计算关系

| 校验式 | 含义 |
|--------|------|
| `[D] = [F] + [G] + [H] + [I] + [J] + [K]` | 风险暴露总和由六类风险暴露构成 |
| `[G] = [G1] + [G2]` | 特定风险暴露 = 资产管理产品 + 资产证券化产品 |
| `[M] = [D] + [L]` | 不考虑风险缓释作用的风险暴露总和 = 考虑风险缓释后的风险暴露 + 风险缓释转出额 |
| `[E] = [D] / [31.D]` | 风险暴露占一级资本净额比例 |
| `[E1] = [D1] / [31.D]` | 不可豁免风险暴露占一级资本净额比例 |

### 校验与勾稽关系

| 类型 | 校验式 | 说明 |
|------|--------|------|
| 表内行校验 | `[0] = [1] + [2] + ... + [30]` | 汇总行等于客户明细行加总，适用于除 A、B、C 外各列 |
