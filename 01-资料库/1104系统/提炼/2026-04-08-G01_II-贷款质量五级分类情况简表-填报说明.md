## 主要内容结构

### 贷款五级分类

| 编号 | 项目 | 说明 |
|------|------|------|
| [1] | 各项贷款 | 包括贷款、贸易融资、票据融资、融资租赁、从非金融机构买入返售资产、透支、各项垫款等 |
| [1.1] | 正常贷款 | 正常类 + 关注类 |
| [1.1.1] | 正常类 | 债务人能够履行合同，无客观证据表明不能按时足额偿付 |
| [1.1.2] | 关注类 | 存在不利影响因素，但目前仍有能力偿付本金、利息或收益 |
| [1.2] | 不良贷款 | 次级类 + 可疑类 + 损失类 |
| [1.2.1] | 次级类 | 无法足额偿付本金、利息或收益，或金融资产已发生信用减值 |
| [1.2.2] | 可疑类 | 已无法足额偿付本金、利息或收益，金融资产已发生显著信用减值 |
| [1.2.3] | 损失类 | 采取所有可能措施后仅能收回极少部分或全部损失 |

### 逾期贷款

| 编号 | 项目 | 说明 |
|------|------|------|
| [2] | 逾期贷款 | 报告期末未按合同规定期限偿还本金或利息的各项贷款本金余额 |
| [2.1] | 逾期60天以上贷款 | 本金或利息逾期时间超过60天的各项贷款本金余额 |
| [2.2] | 逾期90天以上贷款 | 本金或利息逾期时间超过90天的各项贷款本金余额 |

## 关键口径说明

| 项目 | 说明 |
|------|------|
| 五级分类依据 | 正常类、关注类、次级类、可疑类、损失类的定义以《贷款风险分类指引》《商业银行金融资产风险分类办法》为标准 |
| 个人消费贷款逾期 | 按月分期还款的个人消费贷款，逾期90天以内按已逾期部分本金填报；逾期91天及以上按整笔贷款本金填报 |
| 借新还旧逾期起算 | 借新还旧和因借新还旧而展期的贷款，逾期天数从原借款合同到期日起算 |
| 正常展期逾期起算 | 正常贷款展期的逾期天数从展期后到期日起算 |
| 逾期判断口径 | 若本金和利息均逾期，按逾期时间较长的一端判断 |
