## 期限分类规则

| 项目 | 规则说明 |
|------|----------|
| 基本原则 | 按原始（合同）期限分类，不考虑提前支取/还款 |
| 逾期处理 | 超90天非应计贷款仍按原始期限填列 |
| 整数边界 | 含上限不含下限（如"一年至二年"=含二年不含一年） |
| 时间单位 | 月、年按自然月/年统计 |
| 特殊贷款 | <ul><li>贴现/转贴现：贴现日距到期日时间</li><li>分期付款：合同起始日到最终到期日</li><li>分笔发放：按每笔放款日或合同最后到期日</li></ul> |

## 核对关系

### 表内校验关系

| 校验式 | 说明 |
|--------|------|
| `[10] = Σ([1]-[9])` | 存贷款合计 = 九类期限之和 |
| `[A] >= [B]` | 各项存款 >= 储蓄存款 |
| `[11] = [11.1] + [11.2]` | 互联网存款 = 定期 + 活期 |
| `[11] >= [11.a]` | 互联网存款 >= 第三方平台存款 |

### 表间校验关系

| 校验式 | 说明 |
|--------|------|
| `[10.A] = G01_[61.C]` | 各项存款本外币合计 = G01_[61.C] |
| `[10.B] = G01_[27.C]` | 储蓄存款本外币合计 = G01_[27.C] |
| `[10.C] = G01_[62.C]` | 各项贷款本外币合计 = G01_[62.C] |

## 关键口径说明

| 项目 | 说明 |
|------|------|
| **期限定义** | 指各项存款及贷款本金的原始（合同）期限 |
| **通知存款** | 填入"[3.三个月以内]" |
| **互联网存款** | 不通过网点/柜面/开卡机等实体渠道，远程开立II类账户吸收 |
| ** excludes ** | I类户客户关联的II类账户不计入互联网存款 |
| **分笔贷款** | 发放与偿还金额难匹配时，以每笔放款日或合同最后到期日确定期限 |
