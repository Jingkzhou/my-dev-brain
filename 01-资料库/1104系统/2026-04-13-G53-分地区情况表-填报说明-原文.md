# G53《分地区情况表》填报说明-原文

来源：银监统0024号
整理日期：2026-04-13

---

## 一、总体说明

### 1. 报表编码与名称

G53《分地区情况表》，报表文号：银监统0024号。

### 2. 填报机构范围

- **全部填报**：政策性银行、国有大型商业银行、股份制商业银行、城市商业银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资银行、民营银行及其他银行业金融机构
- **金融租赁公司**：仅填报第三部分、第四部分、第六部分
- **货币经纪公司**：仅填报第一部分、第二部分

### 3. 报送范围

法人汇总数据（境内合并报表视角）。

### 4. 报送频率与截止日期

| 部分 | 内容 | 频率 | 截止日 |
|------|------|------|--------|
| 第一部分 | 机构情况简表 | 季报 | 季后13日 |
| 第二部分 | 利润简表 | 季报 | 季后13日 |
| 第三部分 | 贷款行业情况简表 | 季报 | 季后13日 |
| 第四部分 | 五级分类情况简表 | 季报 | 季后13日 |
| 第五部分 | 融资担保公司担保贷款 | 半年报 | 半年后18日 |
| 第六部分 | 县域存贷款 | 季报 | 季后18日 |

### 5. 列结构说明

- 各表按省市自治区（含计划单列市）分列
- **[AG]列**：境内分支机构合计（第一至第五部分）
- **[AH]列**：境外分支机构合计（第一至第五部分）
- 第六部分（县域）：仅境内统计，无境外列

## 二、各部分填报说明

### 第一部分：机构情况简表

填报内容：各地区分支机构数量、从业人员、POS 机具、ATM 机具、资产总额、负债总额。

- 资产总额、负债总额口径：与 G01 报表相关项口径一致
- 境内/境外分支机构按设立地所在省份分列
- 机构数量含本级机构

### 第二部分：利润简表

填报内容：各地区营业收入、营业支出、利润总额、所得税及附加、净利润。

- 口径与 G04 报表相关项口径一致
- 按分支机构所在省份分列

### 第三部分：贷款行业情况简表

填报内容：各地区按国民经济行业分类（GB/T4754）的贷款余额及不良贷款。

- 口径与 G11_I 报表相关项口径一致
- 包含特定区域专栏：
  - 京津冀地区
  - 长江经济带
  - 粤港澳大湾区
  - 自贸区（21个，含上海、广东、天津、福建、辽宁、浙江、河南、湖北、重庆、四川、陕西、海南、山东、江苏、广西、河北、云南、黑龙江、北京、湖南、安徽）

**表间核对关系（第三部分）：**

```
[1.AG] + [1.AH] = G01_[62.C]
```

含义：第三部分行[1.]（各项贷款合计）的境内合计列与境外合计列之和 = G01 第62行C列（各项贷款余额）。

### 第四部分：五级分类情况简表

填报内容：各地区贷款五级分类余额（正常、关注、次级、可疑、损失）及不良贷款率。

- 口径与 G11 报表相关项口径一致
- 不良贷款 = 次级 + 可疑 + 损失
- 按贷款发放地（分支机构所在省份）分列

### 第五部分：融资担保公司担保贷款（半年报）

填报内容：银行业金融机构向融资担保公司提供担保的贷款情况，细分为：

1. 普惠小微企业贷款（融资担保）
2. 政府性融资担保贷款
3. 农户及新型农业经营主体贷款（融资担保）

各项均含：余额、不良余额、当期代偿发生额、当期代偿回收额。

- 半年报：上半年（6月末数据）半年后18日，下半年（12月末数据）半年后18日
- 第五部分含 [AG]（境内合计）/ [AH]（境外合计）

### 第六部分：县域存贷款（季报，季后18日）

填报内容：县域（县、县级市、市辖区中的农村部分）存款、贷款余额及不良贷款。

- 县域范围：依据2020年县域名单
- 细分维度：
  - 已脱贫人口贷款
  - 边缘易致贫人口贷款
  - 已脱贫地区贷款
  - 重点帮扶县贷款
- 存款口径：含单位存款、个人存款，不含同业存款
- 贷款口径：各项贷款，不含票据融资

**表间核对关系（第六部分）：**

```
[1.1AG] + [2.1AG] = G01_[61.C]
```

含义：第六部分行[1.1]（县域单位贷款）和行[2.1]（县域个人贷款）的境内合计列之和 = G01 第61行C列。

## 三、核对关系汇总

### 表内核对

- 各行小计 = 分项之和（通用规则）
- 第三部分：行业分类各行合计 = 各项贷款合计

### 表间核对

| 核对项 | 等式 | 说明 |
|--------|------|------|
| 第三部分贷款总量 | G53_III[1.AG]+[1.AH] = G01[62.C] | 与G01贷款总量勾稽 |
| 第六部分县域贷款 | G53_VI[1.1AG]+[2.1AG] = G01[61.C] | 与G01贷款项目勾稽 |

---

## 四、填报注意事项

1. 境内分支机构按设立注册地所在省份（含计划单列市）分列，不按业务发生地
2. 第三部分贷款行业归属：按贷款最终投向行业，不按借款人注册行业
3. 第五部分为半年报，不随季报一起提交
4. 第六部分仅统计境内县域，无 [AH] 境外列
5. 金融租赁公司跳过第一、二部分；货币经纪公司跳过第三至第六部分
