# 3.7 个人客户关系人

来源：用户提供原文  
性质：监管一表通系统相关原文摘录  
整理日期：2026-04-14

## 采集业务范围

填报个人客户的所有关联关系，包括个人对个人、个人对对公的相关关联关系。本人为本人担保的关系不报送。

## 数据项清单

| 数据项编码 | 数据项名称 | 数据类别 | 数据格式 | 主键 | 数据项解释 | 监管修改记录 |
| --- | --- | --- | --- | --- | --- | --- |
| C070001 | 关系ID | 编码类 | anc..64 |  | 按照既定规则生成，用于唯一区分集团、股东、高管、债务人等人与人、人与物或物与物之间的特定联系的编码。 |  |
| C070002 | 机构ID | 编码类 | anc..24 |  | 按照既定规则生成分配给实际业务经办最小单位机构的唯一编码。 |  |
| C070003 | 个人ID | 编码类 | anc..60 |  | 该个人客户ID，填报同客户ID。 |  |
| C070004 | 社会关系 | 代码类 | 2!n |  | 01-配偶；02-子女；03-父母；00-其他。 |  |

| C070008 | 关系人证件号码 | 编码类 | anc..100 |  | 该关系人所持有效证件号码，需与证件类型配对使用。 |  |

## 备注

- 本表覆盖个人对个人、个人对对公的关联关系。
- 本人作为本人担保的关系不报送。
- `关系人ID` 用于回连个人客户基本情况。
- `社会关系` 主要识别配偶、子女、父母和其他关系。
