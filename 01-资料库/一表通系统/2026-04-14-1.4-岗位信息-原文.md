# 1.4 岗位信息

来源：用户提供文本  
性质：监管一表通系统相关原文摘录  
整理日期：2026-04-14

## 采集业务范围

银行机构内设立的所有岗位的相关信息，至少应包括核心和人力资源系统中的所有岗位信息。

## 数据项清单

| 数据项编码 | 数据项名称 | 数据类别 | 数据格式 | 主键 | 数据项解释 | 监管修改记录 |
| --- | --- | --- | --- | --- | --- | --- |
| A040001 | 机构ID | 编码类 | anc..24 |  | 按照既定规则生成分配给实际业务经办最小单位机构的唯一编码。 |  |
| A040002 | 岗位编号 | 编码类 | an..100 | PK | 填报机构内部的员工岗位编号，同一员工存在多个岗位时，用英文半角分号隔开。 |  |
| A040003 | 岗位种类 | 文本类 | anc |  | 填报机构内部岗位种类的中文描述，外资银行也应翻译为中文描述。 |  |
| A040004 | 岗位名称 | 文本类 | anc..100 |  | 填报机构内部的员工岗位名称，同一员工存在多个岗位时，用英文半角分号隔开。 |  |
| A040005 | 岗位说明 | 文本类 | anc |  | 填报机构内部岗位工作职责的中文描述，外资银行也应翻译为中文描述。 |  |
| A040006 | 岗位状态 | 文本类 | anc |  | 填报机构内部岗位状态的中文描述，外资银行也应翻译为中文描述。 |  |
| A040009 | 是否柜员标识 | 代码类 | 2!n | PK | 01-是；02-否。填报是否柜员。 |  |
| A040007 | 备注 | 文本类 | anc |  | 描述其他字段未能详尽说明的情况，或标注对本条报送记录的特殊说明。不同备注事项用英文半角分号隔开。 |  |
| A040008 | 采集日期 | 日期类 | YYYY-MM-DD |  | 数据表字段信息的采集日期，特定日历日的标识。由日历年、日历月、日历日等数据组合表示。 |  |

## 备注

- 该表强调岗位主数据，不是员工主档的重复。
- `岗位编号` 与 `是否柜员标识` 共同决定岗位是否属于柜员类岗位。
- `岗位种类`、`岗位名称`、`岗位说明` 适合用于岗位字典和职责核对。
