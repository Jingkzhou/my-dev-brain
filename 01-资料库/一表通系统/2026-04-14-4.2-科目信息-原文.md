# 4.2 科目信息

来源：用户提供原文  
性质：监管一表通系统相关原文摘录  
整理日期：2026-04-14

## 采集业务范围

填报机构的所有科目的信息及归属关系，应覆盖所有总账会计全科目涉及的科目。

## 数据项清单

| 数据项编码 | 数据项名称 | 数据类别 | 数据格式 | 主键 | 数据项解释 | 监管修改记录 |
| --- | --- | --- | --- | --- | --- | --- |
| D020001 | 科目ID | 编码类 | anc..32 |  | 按照既定规则生成用于唯一识别一个银行内部会计科目项的编码。同一条数据涉及多个明细科目的，仅填报该笔业务指向的主要科目。 |  |
| D020002 | 机构ID | 编码类 | anc..24 |  | 按照既定规则生成分配给实际业务经办最小单位机构的唯一编码。 |  |
| D020003 | 科目名称 | 文本类 | anc |  | 机构实际使用的总账会计科目编码对应的名称。 |  |
| D020004 | 科目级次 | 代码类 | 2!n |  | 01-一级；02-二级；03-三级；04-四级。总账科目为一级科目。 |  |
| D020005 | 科目类型 | 代码类 | 2!n |  | 01-资产；02-负债；03-所有者权益；04-损益；05-资产负债共同类；06-表外；00-其他。 |  |
| D020006 | 借贷标识 | 代码类 | 2!n |  | 01-借；02-贷；03-借贷并列。 |  |
| D020007 | 归属业务子类 | 代码类 | 2!n |  | 无对应科目的可以为空。同一科目号需填报多个业务子类时，使用英文半角分号“;”分隔。 |  |
| D020008 | 上级科目ID | 编码类 | anc..32 |  | 该科目所属上一级科目，用于构建科目树，如果已经是一级科目，可以允许为空。 |  |

| D020010 | 备注 | 文本类 | anc |  | 描述其他字段未能详尽说明的情况，或标注对本条报送记录的特殊说明。 |  |
| D020011 | 采集日期 | 日期类 | YYYY-MM-DD |  | 数据表字段信息的采集日期，特定日历日的标识。 |  |

## 备注

- 覆盖机构所有科目的信息及归属关系。
- 应与 `4.1 总账会计全科目` 配套，覆盖其涉及的全部科目。
- `归属业务子类` 的取值依赖超链接定义，后续可再单独沉淀规则页。
