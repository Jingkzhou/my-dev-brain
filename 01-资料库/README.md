# 资料库

这里存放原始资料的原文页、摘录页和来源记录。

监管类资料默认按“系统目录直存原文”的方式组织，不再额外增加 `原文/` 子目录。

建议命名：

- `YYYY-MM-DD-标题.md`

当某一类资料持续增长时，可按系统或专题建立子目录，例如：

- `01-资料库/1104系统/`
- `01-资料库/反洗钱系统/`

建议流程：

1. 放入原始资料
2. 从原文提炼核心观点
3. 合并到相关主题页和实体页
