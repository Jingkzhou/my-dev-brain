## 核心规则与结论

### 外汇折算汇率规则

- 四大货币（美元/欧元/日元/港币）：报告期末最后一天国家外汇管理局公布的基准汇价
- 其他货币：以末日美元基准汇率 × 当日上午9时国际外汇市场该货币兑美元汇率套算

### 即期资产/负债（A/B栏）边界规则

- **包含**：已到交割日但未能成功交收的交易
- **不包含**：交易后但未到交割日的交易（T+2惯例）
- **不包含**：结构性敞口头寸
- **不包含**：因表外交易市价重估引致的资产/负债
- 有减值准备的资产以**总额**填报（不净扣损失准备）

### 远期买入/卖出（C/D栏）边界规则

- **包含**：远期外汇合约、外汇期货、外汇掉期、**未到交割日的现货合约**
- **不包含**：货币期权合约（期权在E栏）
- **不包含**：贷款承诺等或有项目（例外：极可能坏账须提专项准备者）
- 回购/返售协议按实质重于形式处理（实质为转移风险则远期记录）

### Delta等值法（E栏）

- 调整后期权头寸 = 合约本金 × Delta
- Delta 由机构专用期权计价模型计算
- 在即期或远期市场对期权头寸进行的套期，计入 A/B/C/D 栏，不计入 E 栏

### 结构性外汇头寸（G/K栏）

- 与《商业银行资本管理办法》定义一致
- 来源：①非并表关联企业外币投资；②并表子公司/分支机构外币投资；③其他符合条件头寸
- 扣除期限至少6个月，存续期内资本处理需保持一致
- **不得**将采用成本法核算的外币长期股权投资对应的附属公司外币留存收益在法人口径下扣除
- **不得**将固定资产、物业、无形资产等资产端项目扣除
- 外币资本金需经金融监督管理总局认可后方可作为结构性头寸扣除

### 总敞口头寸计算（短边法）

1. 分别汇总所有币种的净多头头寸之和、净空头头寸绝对值之和
2. 取两者中较大值作为总敞口头寸
3. [12.F]、[12.J] 结果均 ≥ 0

## 核对关系

| 公式 | 适用范围 |
|------|----------|
| [F] = ([A]-[B]) + ([C]-[D]) + [E] | 适用 1-11 行 |
| [J] = [F] + [H] + [I] | 适用 1-11 行 |
| [12.F] = max(Σ正值F, \|Σ负值F\|) | 短边法合计 |
| [12.J] = max(Σ正值J, \|Σ负值J\|) | 短边法合计 |
| [12.F] ≥ 0，[12.J] ≥ 0 | 必须为非负 |
| [10.J] ≥ 0，[11.J] ≤ 0 | 多头行为正，空头行为负 |

**口径简化条件：**
- 无海外分行/附属机构：J=F，K=G
- 法人汇总口径：不填 I 栏，J=F+H

## 问题索引

| 问题 | 结论 | 来源 |
|------|------|------|
| 即期资产是否包含未到交割日的合约？ | 不包含；T+2未交割前计入远期（C/D栏） | 第三部分（五）1 |
| 期权头寸如何填报？ | E栏填Delta等值（本金×Delta），套期部分计入A/B/C/D | 第三部分（五）3 |
| 外币资产减值准备如何处理？ | 以总额填报，不扣减损失准备 | 第三部分（五）1 |
| 结构性外汇头寸最短扣除期限？ | 至少6个月 | 第三部分（五）5 |
| 黄金是否参与总敞口计算？ | 是，黄金视同外汇，参与短边法计算（仅简化标准法机构） | 第三部分（一） |
| 外资法人银行外国银行分行如何区分口径？ | 外资法人银行法人汇总+合并均报；外国银行分行无法人/合并区别 | 第二部分 3 |
| 总敞口头寸为何不是各货币净敞口的绝对值加总？ | 采用短边法：取净多头总和与净空头总和绝对值中的较大者，反映整体净敞口 | 第三部分（一）/（四） |

## 与其他报表的关联

- **G01_V（主要资产负债项目分币种情况表）**：G32 A/B 栏的外汇资产/负债数据与 G01_V 按币种分列的资产负债存在对应关系；G01_V 反映余额，G32 反映风险敞口（含衍生品）
- **G44（杠杆率情况表）**：G44 衍生品风险暴露采用 SA-CCR 方法，与 G32 远期/期权头寸的衍生品口径有差异（G44 关注信用风险，G32 关注汇率风险）
- **G33_Ⅱ/G33_Ⅰ（银行账簿利率风险）**：同属市场风险计量体系，G32 计量汇率风险，G33 系列计量利率风险

## 规则线索

- 黄金视同外汇，但"仅限简化标准法计量市场风险的填报机构"——待确认是否与资本管理办法中的市场风险简化标准法完全对应
- E栏填报机构范围：仅从事会产生外汇风险的期权交易机构，与是否使用简化标准法不强制关联
