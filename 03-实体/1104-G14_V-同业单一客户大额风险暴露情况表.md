# 1104-G14_V-同业单一客户大额风险暴露情况表

## 简介

G14_V《同业单一客户大额风险暴露情况表》是 1104 系统中的大额风险暴露专项报表部分，核心功能是反映同业单一客户风险暴露明细，并为压力测试要求固定覆盖前100家同业单一客户。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银保监统号（待确认）
- 报表名称：大额风险暴露统计表第 V 部分：同业单一客户大额风险暴露情况表
- 整理日期：2026-04-09
- 来源状态：填报说明原文已入库

## 当前结论

- G14_V 为满足压力测试需求，不论同业单一客户是否达到大额风险暴露标准，都需报满前100家同业单一客户。
- 若大额风险暴露客户不足100个，需补足至前100家风险暴露客户；若超过100个，仅填报前100家大额风险暴露客户。
- 本表按 `[D]考虑风险缓释作用的风险暴露总和` 降序排列，主表客户数量定长100家。
- 本表除主表外，还带有“最大单家及全部同业融资业务情况（扣除结算性同业存款和风险权重为零资产后）”附表，用于反映同业融资集中度。
- 本表的核心比例指标均以一级资本净额为分母，并与 G40 报表存在表间勾稽：`[101.D] = G40_[2.A]`。

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 法人汇总数据为季报（季后18日内）；合并报表数据为半年报（半年后40日内） |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元、百分比（保留两位小数） |

| 类别 | 机构范围 |
|------|----------|
| 适用机构 | 政策性银行（含开发银行）、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行 |
| 不填报本部分机构 | 金融租赁公司、汽车金融公司、消费金融公司、农村资金互助社、贷款公司 |

### 客户筛选与排序规则

| 项目 | 说明 |
|------|------|
| 客户类型 | `[A]` 填写同业单一客户 |
| 入表原则 | 为满足压力测试需求，固定覆盖前100家同业单一客户 |
| 数量限制 | 主表定长100家 |
| 排序要求 | 按 `[D]` 列降序排列 |

### 核心字段关系

| 字段 | 说明 |
|------|------|
| `[D]` | 考虑风险缓释作用的风险暴露总和 |
| `[D1]` | 其中：不可豁免风险暴露 |
| `[L]` | 风险缓释转出的风险暴露（转入为负数） |
| `[M]` | 不考虑风险缓释作用的风险暴露总和 |
| `[101.D]` | 一级资本净额 |
| `[102]` | 最大单家同业融资业务情况 |
| `[103]` | 全部同业融资业务合计 |
