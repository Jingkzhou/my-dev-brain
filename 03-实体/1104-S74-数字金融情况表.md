# 1104-S74-数字金融情况表

## 简介

S74《数字金融情况表》是 1104 监管报表体系中跟踪**数字经济产业贷款**和**金融服务数字化转型**的专项统计表，表号为**金监统S74**（已采用金监统编号，属于较新制定的报表）。依据《数字经济及其核心产业统计分类（2021）》（国家统计局令第33号）对数字经济产业进行五大类分类统计，同时采集客户移动端开通率、金融科技投入、线上业务笔数等数字化转型指标。

## 当前结论

- **金监统编号**：S74 使用金监统编号而非银监统，是1104体系中采用新编号体系的专项统计报表之一
- **境内口径季后13日**：与 S63/S64/S65/S67 系列一致，属于常规季报节奏
- **混合单位**：户、万元、人、笔四种单位混用，是1104体系中单位类型最多的报表之一
- **两部分结构**：第I部分为数字产业贷款（6列×5行层级体系）；第II部分为金融服务数字化转型（客户数量+科技投入+业务笔数）
- **数字经济五大类分类**：依据国家统计局2021年分类标准，01-04大类为核心产业，05大类为效率提升业；两类互斥，一笔贷款不能同时归属
- **A列户数用≤而非=**：[1.A] ≤ [1.1.A]+[1.2.A]，因为同一借款户可能同时跨核心产业和效率提升业，B-F列（金额类）才用等号
- **F列年化利息收益**：仅统计当年发放贷款（非往年）的利息，并按年折算，是独特的时间口径设计
- **强勾稽 G01_VII**：C列（贷款余额）和D列（中长期贷款余额）均与 G01 第VII部分勾稽——这揭示了G01存在第VII部分（数字经济贷款相关）

## 重点规则

### 数字经济产业分类（按国统局2021年标准）

```
数字经济产业 [1.]
├── 数字经济核心产业 [1.1.]
│   ├── 01大类：数字产品制造业 [1.1.1.]
│   ├── 02大类：数字产品服务业 [1.1.2.]
│   ├── 03大类：数字技术应用业 [1.1.3.]
│   └── 04大类：数字要素驱动业 [1.1.4.]
└── 05大类：数字化效率提升业 [1.2.]（与核心产业互斥）
```

**带\*行业的处理**：该行业仅对应国民经济行业分类小类中的部分活动，需根据贷款实际用途逐笔审慎判断。对于 [1.2.]（效率提升业），如无法准确区分，暂不纳入统计。

**互斥规则**：一笔贷款不能同时属于核心产业（1.1.）和效率提升业（1.2.）。

### 贷款余额口径（C列）

```
C列包含（仅单位贷款）：
- 透支
- 经营贷款
- 固定资产贷款
- 融资租赁
- 贸易融资
- 并购贷款
- 从非金融机构买入返售资产
- 各项垫款
- 票据融资（直贴）

注意：
- 仅含单位贷款，不含个人贷款
- 票据融资仅含直贴（不含转贴现）
```

### F列年化利息收益计算

```
计算对象：当年（年初至报告期末）发放的贷款
计算方式：将该贷款已产生的利息收入按年折算

公式示例：
  贷款金额10万元，持有2个月产生利息1000元（0.1万元）
  年化利息收益 = 0.1 × 12/2 = 0.6 万元

排除：往年发放贷款的利息收入不纳入统计
```

### 金融科技投入（第II部分，[4.]）

```
上一年度金融科技投入金额 =
  基础设施建设投入
+ 电子设备采购投入
+ 软件采购投入
+ 系统开发项目投入
+ 系统运维投入
+ 信息科技咨询投入
+ 信息科技人力资源费用
+ 其他投入

注：非一次性投入仅统计上一年度实际投入部分
```

### 表内核对关系

```
行（A列——户数用不等号）：
[1.A] ≤ [1.1.A] + [1.2.A]          同一借款户可能跨多产业，故≤而非=
[1.1.A] ≤ Σ[1.1.1.A~1.1.4.A]      同理

行（B-F列——金额类用等号）：
[1.] = [1.1.] + [1.2.]             B-F列
[1.1.] = [1.1.1.]+[1.1.2.]+[1.1.3.]+[1.1.4.]   B-F列

比率关系：
[2.1.A] ≤ [2.A]     移动端开通≤个人客户总数
[3.1.A] ≤ [3.A]     移动端开通≤法人客户总数
[5.1.A] ≤ [5.A]     线上业务≤全渠道业务

列：
[C] ≥ [D]     贷款余额 ≥ 中长期贷款余额
[C] ≥ [E]     贷款余额 ≥ 不良贷款余额
```

## 规则线索

- **金监统编号揭示报表时间线**：S74 采用"金监统"编号（对应2023年国家金融监督管理总局成立后的新编号），说明 S74 是 1104 体系中相对较新的报表，反映监管机构整合后的政策优先级（数字经济和数字金融）
- **G01_VII 的发现**：S74 与 G01 第七部分（G01_VII）的勾稽关系，揭示了 G01 资产负债项目统计表存在第VII部分，专门统计数字经济贷款；G01 实体页应补充此信息（当前 G01 实体页可能仅记录了其他部分）
- **A列用≤的深层逻辑**：与 S63_III 控股类型表的"H≤A+B+C+D+E+F"逻辑类似，当同一主体可能跨多个维度时，行合计与子项之和存在双重计数问题，监管选择≤而非=来处理这个边界情形
- **F列年化利息收益是政策绩效指标**：不是会计利息，而是通过年化折算消除时间差异，使得季报数据具有可比性；"仅统计当年发放"的设计意图是跟踪新增数字经济贷款的定价水平，而非存量
- **数字化效率提升业（05大类）的审慎原则**：带*行业如无法准确区分暂不纳入，说明监管预期该类统计在初期会有较多不确定性，选择保守口径；相比之下，核心产业（01-04大类）的认定标准更为明确
- **[4.] 金融科技投入是上年数据**：在季报中填报上一年度数据，意味着该指标在全年四个季度中数值不变（除非数据更正），其目的是建立稳定的纵向对比基准，而非追踪季度内的资本支出变化
