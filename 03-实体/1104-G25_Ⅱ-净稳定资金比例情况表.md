# 1104-G25_Ⅱ-净稳定资金比例情况表

## 简介

G25_Ⅱ 是净稳定资金比例（NSFR）计量报表，属于 G25 的第二部分（第一部分 G25_Ⅰ 为流动性覆盖率 LCR）。NSFR 考察银行**未来1年内**稳定资金来源能否覆盖资产和表外业务的资金需求，目标是减少银行对短期批发融资的依赖、降低流动性期限错配风险。

`NSFR = 可用的稳定资金（ASF）/ 所需的稳定资金（RSF）≥ 100%`

## 关键信息

- 所属系统：1104 监管报送系统
- 报表编码：G25_Ⅱ（银监统0027号，G25 第II部分）
- 数据单位：万元；百分比保留两位小数
- 填报币种：本外币合计人民币（标准外币折算规则）
- 报送频度：季报，**法人口径季后18日**；合并口径季后40日
- 适用机构：**大型商业银行（含邮储银行）、股份制商业银行、资产规模2000亿元以上的城商行/民营银行/农商行、外资法人银行**（大中型银行）
- 关联报表：G25_Ⅰ（LCR，流动性覆盖率，同一套表的第I部分）
- 来源状态：已入库
- 整理日期：2026-04-14

## 当前结论

1. NSFR 与 LCR 共用大量概念定义（见"关键概念"段），查询时两表定义联动。
2. 报表为**二维矩阵**结构：横向按负债/资产/表外项目分类，纵向按剩余期限分三档（<6个月/6-12个月/≥1年）× 折算率 × 折算金额。
3. **ASF 与 RSF 的内嵌期权方向相反**：ASF 按最短期限（对银行不利），RSF 按最长期限（对银行不利），均取保守口径。
4. 不良贷款和逾期90天以上资产统一归入 RSF[16.其他资产]，不参与常规折算。
5. **大额存单**行项目归属取决于是否可转让及发行对象。
6. **同业存单**：发行→ ASF[6.2]；持有→ RSF[9.]（不符合 HQLA 口径）。
7. 衍生品净额处理：ASF 侧衍生负债 [I_7] 与 RSF 侧衍生资产 [Ⅱ_11] 通过特殊公式相互轧差，且附加要求为衍生负债的 20%（RSF[13]）。

## 报送口径

### 报表结构

| 部分 | 内容 | 项目数 |
|------|------|--------|
| I. 可用的稳定资金（ASF） | 负债和所有者权益 10 类 | [1.]~[10.] |
| II. 所需的稳定资金（RSF） | 资产和表外项目 17 类 | [1.]~[17.] |
| III. 汇总 | NSFR = ASF/RSF | 自动计算 |

### 列结构（纵向）

| 列 | 含义 |
|----|------|
| A | 剩余期限 < 6个月 金额 |
| B | 剩余期限 6-12个月 金额 |
| C | 剩余期限 ≥ 1年 金额 |
| D/E/F | 各档折算率（% ） |
| G/H/I | 各档折算金额（G=A×D，H=B×E，I=C×F） |
| J | 折算后总额（G+H+I） |

## 重点规则

### 关键概念（与 LCR 共用定义）

以下概念均与 G25_Ⅰ（LCR）保持一致：
- 合格优质流动性资产（HQLA）：一级/2A/2B 资产
- 稳定存款 / 欠稳定存款
- 业务关系存款 / 非业务关系存款
- 小企业客户（存款总额并表口径 ≤ 800 万元且视同零售管理）
- 无变现障碍
- 信用便利 / 流动性便利

### 变现障碍资产定义

下列资产存在变现障碍：
- 作为抵质押物或信用增级工具的资产
- 用于支付经营成本（租金、工资等）的资产
- 规定期间内不得出售/转让的资产（如 ABS 发起机构须持有的 ≥ 5% 最低档次证券）
- 证券融资/抵押品互换中用作抵押的资产

**变现障碍期限** = 对应交易（ABS/担保债券/证券融资交易）的剩余期限。

> 例：卖出回购中用作抵押但未出表的债券，变现障碍期限 = 卖出回购剩余期限。

### 相互依存负债和资产的认定条件

同时满足以下四条：
1. 资产和负债项目可清楚识别
2. 负债与其对应资产的期限和本金数量相同
3. 银行作为转手方，将收到的资金用于相应资产
4. 每对相互依存负债和资产的**对手方不同**

### 大额存单行列归属规则

**列（期限）**：1年内到期或允许提前支取 → E列（<1年）；其他 → F列（≥1年）

**行（分类）**：

| 大额存单类型 | 行项目归属 |
|------------|-----------|
| 不可转让，面向零售/小企业 | [3. 来自零售和小企业客户的融资] |
| 不可转让，面向大中型企业 | [4. 来自大中型企业...的融资] |
| 可转让但个人/机构间不可互转，面向个人 | [3.] |
| 可转让但个人/机构间不可互转，面向机构 | [4.] |
| 个人和机构间均可自由转让 | 统一→ [4.] |

### 同业存单归属

| 方向 | 归入项目 |
|------|----------|
| **发行**同业存单 | ASF [6.2 来自金融机构——非业务关系存款及其他无担保借款] |
| **持有**同业存单 | RSF [9. 不符合 HQLA 定义的证券及其他期限<1年无担保工具] |

### 衍生品特殊处理

```
ASF [I_7. 衍生产品负债净额] = [7.1 衍生负债] − [7.2 提供的变动保证金]
RSF [Ⅱ_11. 衍生产品资产净额] = [11.1 衍生资产] − [11.2 收到的现金变动保证金]

衍生负债净额和衍生资产净额之间的轧差（特殊公式）：
  [I_7.I]  = IF([I_7.C] ≥ [Ⅱ_11.C], ([I_7.C]−[Ⅱ_11.C]) × [I_7.F], 0)
  [Ⅱ_11.I] = IF([Ⅱ_11.C] ≥ [I_7.C], ([Ⅱ_11.C]−[I_7.C]) × [Ⅱ_11.F], 0)

附加要求：RSF[Ⅱ_13.C] = 0.2 × I.可用稳定资金[7.1C]（自动计算，无需填报）
```
