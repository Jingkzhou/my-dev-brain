# 1104-S71-银行业普惠金融重点领域贷款情况表

## 简介

S71《银行业普惠金融重点领域贷款情况表》是监管报表体系中涵盖范围最广的普惠金融专项报表，表号待定，编码为"金监统S71"。含三个子表，覆盖普惠型小微企业（表I，月报）、普惠型涉农（表II，月报）和普惠型消费贷款（表III，季报），构建了银行业普惠金融重点领域贷款的完整统计指标体系。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S71（表号待定）
- 报表名称：银行业金融机构普惠金融重点领域贷款情况表
- 整理日期：2026-04-11
- 来源状态：填报说明原文已入库

## 当前结论

- S71 三表结构频率不同：表I/II 月报（月后6日），表III 季报（季后13日）
- 汽车金融公司、消费金融公司仅填报表III；贷款公司、农村资金互助社需填报（不同于 S72）
- 授信门槛体系四层：3000万（普惠型小微企业法人）→ 1000万（[6]普惠型小微企业贷款）→ 500万（普惠型农户经营性贷款）→ 10万（普惠型消费贷款）
- [6.普惠型小微企业贷款] 为自动加总项（=[1]+[3]），无需手工填报
- 表I 行项目之间存在非加总的"≤"或"可能重叠"关系，不能简单累加，需注意
- [6.3 无还本续贷] 需严格按金规〔2024〕13号认定：原贷款到期前签新合同以新贷结旧贷，不含到期后的重组/展期
- 表间核对跨越最多：与 S63（大中小微型企业贷款）、G11（资产质量五级分类），且表II与表I之间也有等量/不等量核对关系
- [7.不良处置] 统计口径同 G12[15.1B]（G12 尚未入库）
- 涉农贷款存在监管集市 ETL 加工口径：信贷系统涉农贷款分类为基础，对公客户使用 `C_` 拼接，个人客户使用 `P_` 拼接；一体化系统视图将对公中的公司名称个体工商户映射为 `P_`，因此 `S71` 取数时公司名个体工商户按个人报送

### 表II 关键说明

- **农户定义**：长期（一年以上）居住在乡镇（不含城关镇）行政区域内的住户；举家外出谋生一年以上者不属于农户
- **[1.1.2] 为扣减项**：用于扣除农户贷款与个体工商户/小微企业主贷款的重叠数据
- **粮食重点领域**：覆盖从种植（谷物011/豆类0121/薯类0123）→生产资料（肥料262/农药263）→加工（磨制131/淀粉1391/豆制品1392）→批发零售仓储（5111-5221/5951）→农田基本建设的全链条

### 表III 关键说明

- **助学贷款**：= [1.1.1 国家助学贷款] + [1.1.2 生源地助学贷款] + [1.1.3 一般商业性助学贷款]
- **重叠说明**：[1.2 校园消费贷款] 和 [1.3 农户消费贷款] 与 [1.1.3 一般商业性助学贷款] 可能存在数据重叠

## 规则线索

- **"普惠型"门槛体系**：S71 构建了3000万→1000万→500万→10万的四级授信门槛，是整个普惠金融统计的骨架，后续可考虑建立"普惠金融授信口径"规则页
- **[6] 的双重特殊性**：既是自动计算项（不手填），又是门槛更严的指标（1000万 < 3000万），两者差异来自"单户授信1000万以下不含票据融资"的额外限制
- **科技型小微与S70的关联**：[1.2 科技型小微企业] 的认定与 S70 使用相同的"七类科技名单"，S70 尚未入库，待入库后需补充交叉引用
- **不良处置与G12勾稽**：[7] 同 G12[15.1B]，与 S72[5] 同 G12[15.1B] 的逻辑相同，G12 尚未入库
- **粮食重点领域的行业编码**：明确列出了14类 GB/T4754 行业编码，后续可整合到贷款投向规则体系
- **涉农贷款 ETL 规则**：`S71` 的涉农取数应优先看信贷系统分类和一体化视图映射，避免将公司名个体工商户错归对公口径
