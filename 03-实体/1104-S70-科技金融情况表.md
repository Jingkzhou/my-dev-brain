# 1104-S70-科技金融情况表

## 简介

S70《科技金融情况表》是 1104 监管报表体系中的科技金融专项报表，编码"金监统S70"，季报季后13日报送。含四个子表：科技贷款（表I）、投贷联动（表II）、科技金融机构（表III）、科技投资（表IV），是科技金融统计的核心报表，并在 S71、G01_Ⅶ 之间形成上下游核对链路。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S70
- 报表名称：科技金融情况表
- 整理日期：2026-04-11
- 来源状态：填报说明原文已入库

## 当前结论

- S70 四表结构覆盖科技金融"贷款-投资-机构"全链条，季报（季后13日）
- 科技型企业采用**名单制认定**（七类科技名单），优先以中国人民银行提供的企业名单为准
- [1.1] ≤ Σ子类，因汇总时需剔除重复企业（同一企业可能同时符合多类认定）
- [1.2.4.D/E] = G01_Ⅶ[7.A/B]，知识产权密集型产业贷款与贷款投向附注形成等量关系
- [1.1.D/C/G] ≥ S71_I[1.2A/B/C]，科技型企业贷款是科技型小微企业贷款的上位口径
- 表IV的金融资产投资公司、理财公司、信托公司须避免与银行就同一业务重复填报
- 科技支行[5]与科技特色支行[6]互斥：同时满足两类的仅计入[5]
- 数据单位混用：万元（贷款金额）、户（贷款户数）、家（机构数量）、人（人员数）、支（支行数）

## 规则线索

- **科技名单认定的优先顺序**：填报机构自行认定，但"如有，以人民银行提供的企业名单为准"——人民银行名单具有优先效力，后续可关注名单更新频率和获取渠道
- **[1.2.1]高技术制造业 = S72 B列**：两表对高技术制造业的认定使用相同标准（国统字〔2017〕200号），同一企业/贷款的统计口径一致，形成跨报表约束
- **三级统计链路**：G01_Ⅶ[7.A] = S70[1.2.4.D]，S71[1.2A] ≤ S70[1.1.D]，形成"总量→专项→专项中的小微"的三级递进结构
- **投贷联动重复计算风险**：银行与集团内具有投资功能的公司分别填报，同一笔业务有被同时填入[2]和[3]的风险，需关注合并层级的内控
- **[I.买断式转贴现]独立列出**：科技贷款余额[D]不含买断式转贴现，但[I]单独反映此项，提示两口径差异
