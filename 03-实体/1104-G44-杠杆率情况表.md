# 1104-G44-杠杆率情况表

## 简介

G44《杠杆率情况表》是 1104 监管报表体系中的资本充足类报表，依据巴塞尔三框架和《商业银行杠杆率管理办法》设计，计量**一级资本净额与杠杆率总风险暴露**之间的比例。报表文号：银监统0028号。两个口径（法人/合并）分别报送，截止日不同。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：G44
- 报表名称：杠杆率情况表
- 整理日期：2026-04-13
- 来源状态：填报说明原文已入库
- 报表文号：银监统0028号
- 政策依据：巴塞尔三杠杆率框架、《商业银行杠杆率管理办法》

## 当前结论

- **双口径双截止日**：法人机构汇总数据季后18日；合并报表数据季后40日——是1104体系中少数同时要求法人口径和合并口径的报表
- **核心公式**：杠杆率 [6.] = 一级资本净额 [1.] / ([2.]+[3.]+[4.]+[5.])
- **分母四部分**：调整后表内资产 [2.] + 衍生产品资产 [3.] + 证券融资交易资产 [4.] + 调整后表外项目 [5.]
- **证券融资交易轧差**：满足净额结算协议时，[4.2] 可扣减，否则 [4.2]=0；交易对手信用风险暴露 [4.3] 需按 max(0, 应收-应付) 计算
- **表外四档信用转换系数**：可撤销承诺10%；20%档含贸易相关短期或有项目；50%档含一年以上贷款承诺等；100%档含等同贷款的授信
- **衍生产品有效名义本金**：利率/信用类乘监管期限；汇率类取外币较大端；股票/商品类取现价×数量

## 重点规则

### 杠杆率计算结构

```
[6. 杠杆率] = [1.] / ([2.]+[3.]+[4.]+[5.])

[2. 调整后表内资产] = [2.1 表内总资产] - [2.2 衍生产品资产] - [2.3 证券融资交易资产] - [2.4 一级资本扣减项]

[3. 衍生产品资产] = [3.1 重置成本（扣合格保证金）]
                  + [3.2 潜在风险暴露]
                  + [3.3 已扣除的抵质押品]
                  - [3.4 因提供保证金的应收资产]
                  - [3.5 与中央交易对手的清算服务衍生资产]
                  + [3.6 卖出信用衍生产品名义本金]
                  - [3.7 可扣除的卖出信用衍生产品资产]

[4. 证券融资交易资产] = [4.1 会计资产余额（含卖断式）]
                     - [4.2 可扣除余额（净额结算）]
                     + [4.3 交易对手信用风险暴露]
                     + [4.4 代理交易资产余额]

[5. 调整后表外项目] = [5.1 可撤销承诺×10%]
                   + [5.2 20%转换系数表外]
                   + [5.3 50%转换系数表外]
                   + [5.4 100%转换系数表外]
```

### 表外信用转换系数分档

| 系数 | 主要项目 | G4B-2 列号 |
|------|---------|-----------|
| **10%** | 可随时无条件撤销的贷款承诺 | [4.1A]×10% |
| **20%** | 与贸易相关的短期或有项目；原始期限≤1年贷款承诺；符合标准的未使用额度 | [3.C]+[4.2.1C]+[4.3.1C] |
| **50%** | 与交易相关的或有项目；原始期限>1年贷款承诺；信用卡授信额度；票据发行便利；循环认购便利；其他承诺 | [2.C]+[4.2.2C]+[4.3.2C]+[4.4C]+[4.5C]+[4.6C] |
| **100%** | 等同贷款的授信；信用风险仍在银行的资产销售；远期资产购买；远期定期存款；部分交款股票/证券；借出/质押证券；其他表外 | [1.C]+[5.C]+[6.C]+[7.C]+[8.C]+[9.C]+[10.C] |

### 证券融资交易轧差规则

```
满足净额结算协议时：
  [4.2] = [4.1] - max(0, 应收-应付)
  [4.3] = max(0, 全部应收 - 全部应付)

不满足净额结算协议时：
  [4.2] = 0（不得扣减）
  [4.3] 仍需计算
```

**典型案例（见原文 §案例）：**
银行A做回购（持券方）同时做逆回购，满足轧差条件时：
- [4.2] = min([4.1], 应付头寸) = min(300, 500) = 300
- [4.3] = max(0, (525+300)-(500+315)) = 10
- [4.净] = 300-300+10 = 10

### 卖出信用衍生产品有效名义本金

| 衍生品类型 | 有效名义本金计算 |
|-----------|----------------|
| 利率/信用类 | 合约名义本金 × 监管期限（最低10天） |
| 汇率类 | 外币端名义本金（人民币折算后，两端均外币取较大值） |
| 股票/商品类 | 现期单价 × 总数量 |

### 表内核对关系

```
[2.] = [2.1] - [2.2] - [2.3] - [2.4]
[3.] = [3.1] + [3.2] + [3.3] - [3.4] - [3.5] + [3.6] - [3.7]
[4.] = [4.1] - [4.2] + [4.3] + [4.4]
[5.] = [5.1] + [5.2] + [5.3] + [5.4]
[6.] = [1.] / ([2.]+[3.]+[4.]+[5.])
```

### 表间核对关系

```
G44[1.A]  = G4A[8.2]                      （一级资本净额与合格资本表勾稽）
G44[2.1A] = G01[25.C]                     （表内总资产与资产负债表勾稽，仅法人口径）

```

## 规则线索

- **杠杆率与资本充足率的双支柱关系**：G44 是"非风险敏感"指标（分母不加权），与 G4A/G4B 系列的"风险加权"资本充足率互为补充；两类报表共用 G4A 的一级资本净额，G44[1.A]=G4A[8.2] 是跨报表一致性的基础锚点
- **证券融资交易的轧差设计**：[4.2] 可扣减项要求满足净额结算协议，体现监管不鼓励银行通过形式上的净额协议虚减风险暴露；案例中 max(0,应收-应付) 的设计确保永远为非负，交易对手信用风险暴露不能因为应付>应收而归零
- **表外10%系数是监管宽容**：可随时撤销的贷款承诺在资本充足率框架中按100%转换系数计入，但在杠杆率框架中仅按10%，说明巴塞尔三框架对杠杆率与资本充足率的表外处理存在差异，实务中需注意两套规则并行
- **衍生品有效名义本金的监管期限**：对利率/信用类衍生品引入监管期限概念（≥10天），使得短期合约的风险敞口不会被过度低估；期限越长，有效名义本金越大，杠杆率分母越大
- **双口径截止日差异的业务含义**：法人口径季后18日与一般季报一致；合并口径季后40日（与 G52 相同），可能因为合并报表需要汇总子公司数据，处理时间更长
