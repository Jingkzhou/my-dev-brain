# 1104-G01_Ⅵ-托管业务情况表

## 简介

G01_Ⅵ《托管业务情况表》是 1104 系统中的专项报表，用于反映填报机构的托管业务情况。托管是商业银行作为独立第三方，按照法律法规规定和托管合同约定，为所托管产品在中华人民共和国境内提供财产保管及相关服务的行为。

**重要说明**：资金监管业务不属于托管业务，但在本表中列报。

## 当前结论

- G01_Ⅵ 是托管业务专项报表，按托管产品类型分11+1项列示。
- 填报对象不包括政策性银行、农村合作银行、农信用社、村镇银行、民营银行等。
- 重要说明：资金监管业务不属于托管业务，但在本表中列报（[13.]）。
- 私募投资基金托管按净资产余额填报，不按合同金额。

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 月报（境内分支机构汇总），月后6日 |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元（保留两位小数） |

### 填报机构

大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、农村商业银行、外资法人银行、外国银行分行、信托公司。

> **排除**：政策性银行、农村合作银行、农村信用合作社、村镇银行、民营银行、直销银行、企业集团财务公司、金融租赁公司、汽车金融公司、消费金融公司

### 重要说明

| 项目 | 说明 |
|------|------|
| 资金监管业务 | 不属于托管业务，但在本表[13.]中列报 |
| 私募基金统计 | 按净资产余额填报，不按合同金额（保险私募、QDLP/QDIE、QFLP不包含） |
| 保险资管产品 | 包括组合类产品、保险债权计划、保险股权计划、保险私募基金、保险资产支持计划 |
| 养老金产品统计 | 必须向人社部产品备案 |

## 规则线索

### 私募投资基金托管口径

| 细分项 | 说明 |
|--------|------|
| 统计口径 | 填报净资产余额，而非合同金额 |
| 排除范围 | 保险私募基金托管、QDLP/QDIE产品托管、QFLP产品托管 |
| 包含范围 | 私募证券、私募股权、其他私募（如资产配置类） |
| 基金结构 | 契约型、合伙型、公司型基金 |

### 保险资产托管口径

| 项目 | 说明 |
|------|------|
| 统计范围 | 保险集团/公司资金运用、保险资管产品 |
| 包含产品 | 组合类产品、保险债权计划、保险股权计划、保险私募基金、保险资产支持计划 |
| [7.1]统计 | 仅限银保监会指定机构备案的私募基金 |

### 养老金托管口径

| 细分项 | 统计口径 |
|--------|----------|
| [8.1]企业年金 | 已获取人社部门企业年金计划确认函的各类计划 |
| [8.2]职业年金 | 已获取人社部门职业年金计划确认函的职业年金计划 |
| [8.3]养老金产品 | 已向人社部产品备案的养老金产品 |

### QDII/QFI类产品口径

| 类别 | 说明 |
|------|------|
| QDII | 应包括QDII基金、QDII专户、QDII理财、QDII保险、QDII信托 |
| QFII | 包括QFII、RQFII、QFLP、境外机构直接投资银行间债券市场、GDR存托人、GDR跨境转换机构 |
