# 1104-S64-大中小微型企业贷款分行业情况表

## 简介

S64《大中小微型企业贷款分行业情况表》是 1104 监管报表体系中企业贷款分行业专项统计报表，表号为金监统S64。报表以国民经济行业分类（GB/T 4754-2017）为行维度，分两部分反映报告日各类企业贷款的行业分布及其中的不良贷款情况，是 S63/S64/S65 三表联动框架中的行业维度表。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S64
- 报表名称：大中小微型企业贷款分行业情况表
- 整理日期：2026-04-13
- 来源状态：填报说明原文已入库（251版）

## 当前结论

- S64 是"企业贷款行业分布表"，行维度按 GB/T 4754-2017 国民经济行业分类，共列 [1.1]~[1.20] 共 20 个行业大类，加总为 [1]
- 含两部分：第I部分为大中小微型企业贷款分行业余额，第II部分为大中小微型企业不良贷款分行业情况
- 均为季报，季后13日（季报第一批次）报送；境内汇总数据
- 数据单位万元，金额保留两位小数，本外币合计人民币填报
- **贷款口径宽于一般理解**：包含贷款、贸易融资、贴现、融资租赁、买入返售资产、透支、各项垫款；**明确排除**买断式转贴现和买断其他票据类资产
- 普惠型小微企业贷款 = 单户授信总额1000万元及以下的小微企业贷款（含小微法人、个体工商户、小微企业主经营性贷款）
- 关停企业对应的个人经营性贷款（含个体工商户贷款、小微企业主贷款）**不填报**
- 新设立企业无法划型时暂划入中型企业

## 重点规则

### 贷款口径定义

贷款口径 = 贷款 + 贸易融资 + 贴现 + 融资租赁 + 从非金融机构买入返售资产 + 透支 + 各项垫款

**明确排除：**
- 买断式转贴现（从其他金融机构买断未到期已贴现商业汇票，无回购协议）
- 买断其他票据类资产（从其他金融机构买断除商业汇票外其他票据类信贷资产，含福费廷转让、已议付信用证转让等，无回购协议，与是否有追索权无关）

### 企业规模划型

- 划型依据：工信部联企业〔2011〕300号，参考国家统计局最新标准
- 划型适用行业按企业主要经济活动确定
- 企业法人按标准执行，企业非法人按授权法人执行
- 新设立企业（含建设期项目公司）：按可行性研究报告预计数据划型；无法划型时暂划入中型企业
- 应随时跟踪企业情况并及时调整划型

### 行业分类规则

- 标准：GB/T 4754-2017 国民经济行业分类
- 单一经济活动：按该活动确定行业
- 多种经济活动：按增加值份额最大的主要活动确定；增加值难确定时，可按销售收入、营业收入或从业人员确定

### 首贷户定义

首贷户 = 从银行业金融机构首次获得贷款的客户，判定标准：填报前通过人行征信系统查询，该客户无征信记录。

- 当年首贷户首次贷款金额：填报机构当年对首贷户**首次**发放的贷款金额合计；首次贷款结清后再次发放的不计入
- 涉农首贷户：注册地位于农村区域，或注册地位于城市但贷款用于农林牧渔业及支农用途的客户

## 规则线索

- **S63/S64/S65 三表联动**：S63 按企业规模汇总，S64 按行业切片，S65 按地区切片，三表总量行一致（均等于S63_I[1]）
- **E列普惠上限约束**：S64 的 E 列（普惠型小微企业贷款）受 S71 的 A0+3.A0 约束，说明两张报表在普惠口径上存在强关联，填报时需先对齐 S71
- **贷款口径与 G01_VII 的上限关系**：S64 各企业规模列之和不超过 G01_VII 各项贷款总量，说明 S64 是在 G01 基础数据上的再分类，不能超出资产负债表范围
- **买断业务的口径细节**：买断式转贴现和买断其他票据类资产均排除在外，且原文明确说明"与是否有追索权无关"，是常见口径误区
- **F列涵盖关系**：[F.] ≥ [G.]+[H.]，说明 F 列是更宽口径，G、H 列是其中两类子集
