# 1104-S26-城商行省外经营情况统计表

## 简介

S26《城商行省外经营情况统计表》是 1104 监管报表体系中**专门针对城市商业银行**跨区域经营监测的专项报表，依据银保监办发〔2022〕74号建立。分为两个子表：**S26_I**（省外经营情况总量表，按省份分布）和 **S26_II**（省外分支机构明细表）。填报机构仅限城商行。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：S26_I / S26_II
- 报表名称：城商行省外经营情况统计表 / 城商行省外分支机构情况表
- 整理日期：2026-04-13
- 来源状态：填报说明原文已入库
- 政策依据：银保监办发〔2022〕74号

## 当前结论

- **填报机构唯一性**：全 1104 体系中仅 S26 为城商行专属报表，其他报表均含全行业
- 季报，季后18日（**第二批次**，比一般季报的第一批次晚5天）
- **省外/省内判定关键规则**：计划单列市视为所在省份省内，而非省外；例如浙江城商行在宁波的分支机构 = 省内
- **融资投向省份**：以资金**最终使用地**为准；无法确认时以融资企业**注册地**为准
- **重点监测口径**（最严格）= 省外融资中，排除贸易融资、供应链融资、债券投资、理财投资后的**非金融企业**融资
- 理财投资余额仅含**母行**理财，不含理财子公司
- 小微企业贷款口径：单户授信1000万元（含）以下，**不含票据融资**（比普通普惠口径多排除了票据）
- S26_I 列 A/B/C 分别对应 S26_II 列 P/Q/R（省外合计），两表相互勾稽

## 重点规则

### 省外/省内判定规则（计划单列市处理）

计划单列市（宁波、大连、青岛、深圳、厦门）在行政上属于所在省（浙江、辽宁、山东、广东、福建），因此：
- 浙江城商行 → 宁波分支机构 = **省内**
- 广东城商行 → 深圳分支机构 = **省内**

若在其他省份的城市开展业务 = 省外。

### 三类融资口径的层次关系

```
省外融资（最宽）
└── 包含：贸易融资 + 供应链融资 + 债券投资 + 理财投资 + 重点监测口径省外融资
        
重点监测口径省外融资（最严）
= 省外融资 - 贸易融资 - 供应链融资 - 债券投资 - 理财投资
= 省外非金融企业融资中的"纯信贷"部分
```

注：重点监测口径是监管实际关注的跨区域蔓延风险的核心指标。

### 表内核对关系

```text
S26_I：
[1.] = [1.1.1]+[1.1.2]+[1.1.3]+[1.1.4] = [1.3.1]+[1.3.2]
[2.] = [2.1.1]+[2.1.2]+[2.1.3]+[2.1.4] = [2.3.1]+[2.3.2]
[3.] = [3.1.1]+[3.1.2]
     = [3.2.1]+[3.2.2]+[3.2.3]+[3.2.4]
     = [3.3.1]+[3.3.2]
     = [4.1]+[4.2]+[4.3]+[4.4]+[4.5]

[1.A] = [33省外合计.A]   （行[1]的A列 = 33个省外省份合计）
[2.A] = [33省外合计.B]
[3.A] = [33省外合计.C]
```

含义：[1.] 的多维分解（4分项 + 2分项）都指向同一总量，是一个多维度展开的交叉验证设计。

### 表间核对关系

```text
S26_I[3.A] = S26_II[省外合计.P]
S26_I[3.B] = S26_II[省外合计.Q]
S26_I[3.C] = S26_II[省外合计.R]
```

含义：S26_I 的省外合计某列 = S26_II 对应列的所有分支机构之和，两表从总量与明细两个视角交叉验证。

## 规则线索

- **城商行定位管理的监管工具**：S26 是 74 号文的直接配套统计，是监管层用来量化城商行"是否坚守定位"的核心数据来源，填报数据直接影响监管评价
- **重点监测口径 = 监管真正关注的扩张风险**：通过剥离贸易融资、供应链融资等政策鼓励类业务，将"需要管控的跨区域扩张"单独提出；理解这一口径设计是填报合规的关键
- **计划单列市处理是常见误区**：实务中容易将宁波/深圳等计划单列市视为独立区域而误报为省外，S26 明确将其纳入所在省的省内统计
- **第二批次上报**：暗示监管审核排序较晚，可能与季报稳定性要求或城商行基础数据质量有关
- **S26_I 多维分解的验证意义**：行[1.]和[2.]各有两套等式（4分项 + 2分项），意味着贷款和存款都需从两个维度同时保证合计一致，是内部数据质量验证点
