# 1104-S67-房地产融资风险监测统计表

## 简介

S67《房地产融资风险监测统计表》是 1104 监管报表体系中的房地产融资专项报表，编码"金监统S67"，月报月后6日报送。依据《商业银行房地产贷款风险指引》及《商业银行压力测试指引》制定，含两大部分（涉房融资和租赁住房贷款），列维度覆盖余额、当月流量、五级分类、展期、逾期等15列，是 S 系列中列维度最丰富的报表。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S67
- 报表名称：房地产融资风险监测表
- 整理日期：2026-04-11
- 来源状态：填报说明原文已入库

## 当前结论

- S67 是列维度最丰富的 S 系列报表（15列 A-O），将余额、流量、资产质量、展期、逾期合并在同一张表，支持对房地产融资的存量结构+动态变化+风险暴露的一体化监控
- 两大部分：第一部分（涉房融资：贷款+表内非信贷资金+非保本理财+以房产为抵押贷款）、第二部分（租赁住房贷款4类）
- 填报机构含**住房储蓄银行**，不含信托公司、理财公司、金融租赁公司、汽车金融公司
- 核心列等式：**A = F + G + H**（余额 = 正常 + 关注 + 不良），**K = M + N + O**（逾期按1-60天/61-90天/91天以上分档）
- [D.当月处置不良贷款] 与 G12[15.1不良贷款处置总额] 形成等量关系（G12 尚未入库）
- 房产开发贷款[1.2]的范围排除：厂房、自建自用的酒店/仓储、非营利性医院/学校/养老机构
- "保交楼"专项借款和专项借款配套融资统一归入[1.2.1住房开发贷款]
- 逾期填报规则：个人住房贷款逾期90天以内按已逾期本金余额填报，逾期90天以上按整笔贷款本金余额填报

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 月报，月后6日 |

### 适用机构

政策性银行（含开发银行）、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社和村镇银行、**住房储蓄银行**、外资法人银行、外资银行分行、直销银行。（不含金融租赁、信托、理财公司、汽车金融）

### 列维度（A-O）

| 列 | 内容 | 说明 |
|----|------|------|
| **A** | 期末余额 | 贷款本金余额 |
| **B** | 当月新发放金额 | 报告期内新发放 |
| **C** | 当月收回金额 | 现金回收/核销/以物抵贷/卖断等 |
| **D** | 当月处置不良贷款 | = G12[15.1不良贷款处置总额] |
| **E** | 当月核销 | [D] 中的核销金额 |
| **F** | 正常类 | A=F+G+H |
| **G** | 关注类 | 同上 |
| **H** | 不良贷款 | 同上 |
| **I** | 展期余额 | 已展期未到期余额 |
| **J** | 展期变动 | 当月新增/减少 |
| **K** | 逾期合计 | 不含已展期未到期；K=M+N+O；K≤A |
| **M** | 其中：1-60天 | - |
| **N** | 其中：61-90天 | - |
| **O** | 其中：91天以上 | - |

## 重点规则

### 核心列等式

```text
[A] = [F] + [G] + [H]          （余额 = 五级分类合计，适用指定行）
[K] = [M] + [N] + [O]          （逾期合计 = 三档之和）
[K] ≤ [A]                      （逾期 ≤ 余额）
```

### 关键行加总（适用 A-O 列）

```text
第一部分：
[1] = [1.1]+[1.2]+[1.3]+[1.4]+[1.5]
[1.2] = [1.2.1]+[1.2.2]+[1.2.3]
[1.3] = [1.3.1.1]+[1.3.1.2]    （新建+二手）
[1.4] = [1.4.1.1]+[1.4.1.2]+[1.4.1.3]  （按套数：首套/二套/三套以上）
[2] = [2.1]+[2.2]

第二部分：
[1] = [1.1]+[1.2]+[1.3]+[1.4]+[1.5]
```

### 表间核对

```text
[D] = G12[15.1不良贷款处置总额]    （G12 尚未入库）
```

### 逾期填报特殊规则

```text
个人住房贷款和个人商业用房贷款：
  逾期 < 90天 → 按已逾期部分本金余额填报
  逾期 ≥ 90天 → 按整笔贷款本金余额填报
  
借新还旧 → 逾期天数从原借款合同到期日起算
展期贷款 → 逾期天数从展期后到期日起算
本金和利息都逾期 → 按逾期时间较长的一端填报
```

## 规则线索

- **S67 vs S72 的列维度差异**：S72 只有5列（余额/户数/不良/累放/年化利率），S67 有15列还包含了流量（新发放/收回/处置核销）、五级分类、展期、逾期三档，体现"房地产融资是高风险重点监控领域"的监管意图
- **表内非信贷和非保本理财的穿透原则**：[2]和[3]均需按穿透底层资产原则填报，说明监管要求不受金融产品外壳限制，直接看最终投向是否为房地产
- **"保交楼"在第一部分[1.2.1]而非单独行**：政策专项贷款汇入常规开发贷款维度，可能造成[1.2.1]余额结构变化，分析时需注意
- **G12 的双重关联**：S72[5]和S67[D]均与G12[15.1]勾稽，G12 是连接所有专项报表不良处置数据的枢纽报表，其入库优先级较高
