# 1104-S63_III-各控股类型企业融资情况表

## 简介

S63_III《各控股类型企业融资情况表》是 S63 系列的第III部分，报表编码为金监统S63_III。报表以**企业控股类型**（国有/集体/私人/港澳台商/外商/其他）为列维度，反映报告日各类企业贷款按控股性质的分布，是 S63_I（按企业规模）的控股类型视角补充表。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S63_III
- 报表名称：企业贷款情况表（按企业性质划分）
- 整理日期：2026-04-13
- 来源状态：填报说明原文已入库

## 当前结论

- S63_III 是 S63_I 的控股类型维度切片：列 A-F（六类控股类型）+ 列 G（个人经营性贷款）+ 列 H（合计）
- 月报，月后6日，与 S63_I 频率一致
- **关键勾稽确认 S63_I 列结构**：[1.G]=S63_I[1.E]，即 S63_III 第 G 列（个人经营性贷款）= S63_I 第 E 列 → 确认 S63_I 的 A=大型、B=中型、C=小型、D=微型、E=个人经营性贷款
- S63_III 控股 A-F 合计 = S63_I 大中小微四列合计（A+B+C+D），不含个人经营性贷款
- 表外授信（银行承兑汇票/信用证/保函/贷款承诺/PPP贷款）**不含 G 列**，H=A+B+C+D+E+F（无个人经营性贷款）
- 逾期贷款分 **4 档**（60天以内/61-90天/91-360天/361天以上），比 S63_I 的 7 档更粗
- 中长期贷款 [1.4.1] 不含买断式转贴现**和买断其他票据类资产**（与 S64 口径一致）
- 含知识产权质押贷款（户数+金额）和 PPP 贷款两类 S63_I 没有的专项维度
- C1（私人控股-小微企业）= 私人控股中小型+微型之和

## 重点规则

### 列结构（控股类型）

| 列 | 含义 |
|----|------|
| A | 国有控股（绝对控股>50% 或 相对控股/协议控股） |
| B | 集体控股 |
| C | 私人控股，其中 C1=小型+微型 |
| D | 港澳台商控股 |
| E | 外商控股 |
| F | 其他（信托计划等非法人机构控股） |
| G | 个人经营性贷款（对应 S63_I 第E列） |
| H | 合计：贷款类=A+B+C+D+E+F+G；表外授信及PPP=A+B+C+D+E+F |

**控股类型判定规则：**
- 出资比例 >50%：绝对控股
- 出资比例未超50%但最大：相对控股
- 出资比例与另一方各占50%且有国有成分：按国有控股
- 两方各占50%且另一方为非国有：按相应成分控股

### 贷款口径

贷款 = 贷款、贸易融资、贴现、融资租赁、从非金融机构买入返售资产、透支、各项垫款等  
**不含买断式转贴现**（口径比 S63_I 更严：S63_I 是"项目1-13不含"，S63_III 是直接"不含"）

中长期贷款[1.4.1]：不含买断式转贴现**和买断其他票据类资产**（与 S64 完全对齐）

### 逾期贷款分档（4档，比 S63_I 粗）

```text
[1.3.1] 逾期60天以内
[1.3.2] 逾期61天到90天
[1.3.3] 逾期91天到360天
[1.3.4] 逾期361天以上
```

注：S63_I 细化为7档（30天/31-60天/61-90天/91-180天/181-270天/271-360天/361天以上），S63_III 仍保留4档旧口径。

### 知识产权质押贷款

- [3.a] 以知识产权为质押的户数（适用H列）
- [4.a] 当年累计发放知识产权质押贷款（适用H列）
- 知识产权非唯一担保方式时，按最新估值对贷款余额按比例拆分（拆分规则同G13）

### PPP 贷款 [12]

- 按期末余额填报
- [12.1] 新机制前发放 + [12.2] 新机制后发放（依据国办函〔2023〕115号）
- 适用于H列（=A+B+C+D+E+F，**不含G列**个人经营性贷款）

## 规则线索

- **S63_I 列结构的"钥匙"**：S63_III 通过表间核对关系明确了 S63_I 的 A-E 列含义：A=大型、B=中型、C=小型、D=微型、E=个人经营性贷款；Query S63_I 列含义时优先来此核对
- **控股类型与企业规模的正交视角**：S63_I 横向（列）= 企业规模，S63_III 横向（列）= 控股类型，两表共享同一行结构，在数据比对时可互为验证
- **表外授信不统计个人经营性贷款**：银行承兑汇票/信用证/保函/贷款承诺的H列=A-F之和，G列（个人经营性贷款）不参与表外统计，说明表外授信口径仅针对法人企业
- **逾期档次口径差异**：S63_III 仍用4档旧口径，与S63_I的7档不一致，跨表对比逾期数据时需注意档次映射
- **知识产权质押 + PPP 是 S63_I 没有的监测维度**：这两类只在 S63_III 中统计，说明监管在控股类型维度下额外关注的风险点
