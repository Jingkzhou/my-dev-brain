# 1104-G52-地方政府融资平台及支出责任债务持有情况统计表

## 简介

G52《地方政府融资平台及支出责任债务持有情况统计表》是 1104 监管报表体系中专门统计银行业金融机构持有**地方政府融资平台债务**及**政府长期支出责任融资**的报表。填报以地方政府复核结果为前提——**未接到地方政府复核要求的机构，报送空表**。报表文号：银监统 号（原文未注明具体编码）。

## 当前结论

- **填报前提：地方政府复核**：未接到地方政府提出的融资数据复核要求的机构，报送空表即可；这是1104体系中唯一以"地方政府复核结果"为填报触发条件的报表
- **季后40日**：比一般季报（季后13日）晚27天，是1104体系中截止日最晚的季报，原因可能是需等待地方政府完成复核
- **三类融资结构**：
  1. 对融资平台融资（地方政府承诺偿还）
  2. 对融资平台融资（地方政府提供担保）
  3. 形成地方政府长期支出责任的融资（PPP/政府购买服务/政府投资基金）
- **机构类型影响填列范围**：银行填 E-Q 列；信托公司填 R 列（需扣除最终来源于银行的部分）；融资租赁公司填 S 列
- **债务人层级三分**：省级（含计划单列市归省级）/ 市级（含直辖市区、省会城市）/ 县级（含地市级区）
- **行数 = 36省市 + 合计行 + 三类融资行**：1-36行为各省市，37行全国合计，38-40行分三类融资；两套行合计均等于37行
- **贷款 E 列有三套平行等式**：按资产质量（T-Y）、按剩余期限（Z-AC）、按行业投向（AD-AX）三个维度展开，且均须等于 E 列贷款总量

## 重点规则

### 三类融资的定义与优先级

```
第一类：融资平台融资（政府承诺偿还）
第二类：融资平台融资（政府提供担保）
第三类：形成长期支出责任（PPP、政府购买服务、政府投资基金中约定回购/保底收益）

难以区分规则（优先级）：
- 第一类 vs 第二类 难以区分 → 归入第一类
- 第三类 vs 第一类或第二类 难以区分 → 归入第三类
```

### 债务人层级判定

```
省级：省/自治区/直辖市 及其部门设立；计划单列市设立 → 归省级
市级：省辖市/地州政府及其部门；直辖市的区/省会城市设立 → 归市级
县级：县级（含县级市）及以下；地市级的区政府设立 → 归县级
```

### 理财资金口径（M列）

银行理财资金含直接投资和通过信托计划、证券/保险/基金/资管计划间接投资两种方式，均纳入 M 列统计；非标准化债权资产口径依据银发〔2018〕106号。

### 贷款 E 列三套展开等式

```
E = T+U+V+W+X           （五级分类：正常+关注+次级+可疑+损失）
E = Z+AA+AB+AC          （剩余期限：四个期限区间）
E = AD+AE+AG+...+AX     （行业投向：与地方政府协商认定的行业分类）
```

三套等式各自独立，E 列总量须同时满足全部三套分解。

### 表内核对关系

```text
行：
[1.]+[2.]+...+[36.] = [37.]     适用于全部列（A至AX）
[38.]+[39.]+[40.] = [37.]       适用于全部列（A至AX）

列：
[A] = [B]+[C]+[D]               债务人层级合计（1-37行）
[A] = [E]+[F]+[K]+[L]+[M]+[R]+[S]  资金来源合计（1-37行）
[E] = [T]+[U]+[V]+[W]+[X]       贷款资产质量（1-37行）
[E] = [Z]+[AA]+[AB]+[AC]        贷款剩余期限（1-37行）
[E] = [AD]+[AE]+[AG]+[AH]+[AI]+[AJ]+[AL]+[AM]+[AN]+[AO]+[AP]+[AQ]+[AR]+[AS]+[AT]+[AU]+[AV]+[AW]+[AX]  贷款行业投向（1-37行）
```

### 表间核对关系

```text
G52[37.E] ≤ G01[62.C]
```

含义：G52 全国合计中的贷款余额 ≤ G01 各项贷款总余额。注意为不等式（≤），因为并非所有贷款都涉及地方政府融资平台。

## 规则线索

- **空表填报机制是监管设计亮点**：G52 以"地方政府复核"为触发前提，意味着监管数据有地方政府的背书，而非银行单方面认定；未收到复核要求即报空表，避免银行自行判断是否属于融资平台带来的口径混乱
- **季后40日揭示数据复杂性**：截止日远晚于一般季报，说明该数据需要银行与地方政府完成多轮复核确认，填报压力集中在季末后的第二个月
- **三类融资的层次设计**：第一类（承诺偿还）和第二类（提供担保）区分政府责任强度；第三类（PPP等）覆盖隐性债务的另一来源；难以区分时的归并规则（第三类优先）体现监管对隐性债务的保守口径偏好
- **信托 R 列的"穿透扣减"**：信托公司填报前需穿透至最终资金来源，将银行资金部分扣除，避免银行通过信托通道形成的敞口被重复计算
- **E 列三套等式 = 三维风险画像**：一张 E 列贷款数据同时展开为资产质量、期限结构、行业投向三个维度，构成对融资平台贷款的立体风险监测体系
