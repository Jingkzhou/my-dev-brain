# 1104-G14_VI-同业集团客户大额风险暴露情况表

## 简介

G14_VI《同业集团客户大额风险暴露情况表》是 1104 系统中的大额风险暴露专项报表部分，核心功能是反映同业集团客户中达到大额风险暴露门槛的客户明细情况。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银保监统号（待确认）
- 报表名称：大额风险暴露统计表第 VI 部分：同业集团客户大额风险暴露情况表
- 当前资料依据：[[01-资料库/1104系统/提炼/2026-04-09-G14_VI-同业集团客户大额风险暴露情况表-填报说明|G14_VI 同业集团客户大额风险暴露情况表填报说明]]

## 当前结论

- G14_VI 只填报“考虑合格风险缓释作用后风险暴露超过一级资本净额 2.5%”的同业集团客户。
- 本表最多填报前 30 家客户；若超过 30 家，按 `[D]考虑风险缓释作用的风险暴露总和` 降序截取前 30 家。
- 本表不是独立口径体系，多个字段定义分别继承自 G14 的第 I、II、III、V 部分。
- 本表的核心比例指标均以一级资本净额为分母，并与 G40 报表存在表间勾稽：`[31.D] = G40_[2.A]`。

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 法人汇总数据为季报（季后18日内）；合并报表数据为半年报（半年后40日内） |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元、百分比（保留两位小数） |

### 适用机构

| 类别 | 机构范围 |
|------|----------|
| **适用机构** | 政策性银行（含开发银行）、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行 |
| **不填报本部分机构** | 企业集团财务公司、金融租赁公司、汽车金融公司、消费金融公司、农村资金互助社、贷款公司 |

### 客户筛选与排序规则

| 项目 | 说明 |
|------|------|
| 客户类型 | `[A]` 填写同业集团客户 |
| 入表门槛 | 考虑合格风险缓释作用后风险暴露超过一级资本净额 2.5% |
| 数量限制 | 最多填报 30 家客户 |
| 排序要求 | 按 `[D]` 列降序排列 |

### 核心字段关系

| 字段 | 说明 |
|------|------|
| `[D]` | 考虑风险缓释作用的风险暴露总和 |
| `[D1]` | 其中：不可豁免风险暴露 |
| `[L]` | 风险缓释转出的风险暴露（转入为负数） |
| `[M]` | 不考虑风险缓释作用的风险暴露总和 |
| `[31.D]` | 一级资本净额 |

## 重点规则

### 表内校验关系

```text
[0] = [1] + [2] + ... + [30]

[M] = [D] + [L]
[E] = [D] / [31.D]
[E1] = [D1] / [31.D]
[1.D] ≥ [2.D] ≥ [3.D] ≥ ... ≥ [30.D]
```
