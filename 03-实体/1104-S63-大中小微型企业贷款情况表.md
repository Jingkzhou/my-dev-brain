# 1104-S63-大中小微型企业贷款情况表

## 简介

S63《大中小微型企业贷款情况表》是 1104 监管报表体系中企业贷款的**核心总量表**，表号为金监统0033号。报表以企业规模（大/中/小/微型/个人经营性）为列维度，反映报告日各类企业贷款的余额、五级分类、担保方式、逾期情况及多类专项统计，是 S63/S64/S65 三表联动框架的上游锚点。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统0033号（S63）
- 报表名称：大中小微型企业贷款情况表
- 整理日期：2026-04-13
- 来源状态：填报说明原文已入库（261版）

## 当前结论

- S63 是企业贷款三维框架（规模/行业/地区）的总量表：S63 按**企业规模**汇总 → S64 按**行业**切片 → S65 按**地区**切片，S64/S65 总量行均等于 S63_I[1]
- 两个部分频率不同：**第I部分月报**（月后6日），**第II部分季报**（季后13日）；是 S63/S64/S65 三表中唯一含月报的
- 列维度：A=大型、B=中型、C=小型、D=微型、E=个人经营性贷款；[E.]≥[F.]+[G.]（F/G为其中子项）
- **贷款口径的关键差异**：S63 项目1-13 不含买断式转贴现，但买断式转贴现作为独立项目[14/29]单独统计；S64 则将买断式转贴现和买断其他票据类资产全部排除在贷款口径之外
- 企业定义：含有限责任公司、股份有限公司、合伙企业、个人独资企业等；**不含**行政机关、事业单位、社会团体、个体工商户（个体工商户通过"个人经营性贷款"口径纳入）
- 个人经营性贷款：含个体工商户贷款、小微企业主贷款、农户生产经营贷款；小微企业主 = 法定代表人及配偶、主要股东、实际控制人
- 逾期分类：细化为7档（30天以内/31-60/61-90/91-180/181-270/271-360/361天以上）
- 专精特新中小企业：工信部认定的"小巨人"+省级认定的"专精特新"，与 S70 保持一致
- 当年累放数据遵循"划型变更时余额等全量切换，累放数据在原分类留存"原则

## 重点规则

### 贷款口径

S63 贷款 = 贷款、贸易融资、贴现、融资租赁、从非金融机构买入返售资产、透支、各项垫款等

**关键限定：项目1-13 不含买断式转贴现**（买断式转贴现作为独立项目[29/14]统计）

与 S64 口径差异：
- S63：买断式转贴现单独统计，不含在项目1-13内
- S64：买断式转贴现 + 买断其他票据类资产 **全部排除**在贷款口径之外

### 企业划型

- 划型依据：工信部联企业〔2011〕300号，参考国家统计局最新标准
- 停产清算中企业：参考停产前指标划型（S64中无此条款，S63特有）
- 划型变更规则：余额、余额户数、不良贷款等**全量切换**至新分类；累放数据在原分类留存

### 贸易融资口径

贸易融资 = 打包贷款、进口押汇、出口押汇、议付信用证、福费廷等  
**不包含**：本表中贴现[1.2.4]和买断式转贴现[14]

### 逾期贷款7档分类

```text
逾期30天以内 / 31-60天 / 61-90天 / 91-180天 / 181-270天 / 271-360天 / 361天以上
```

### 无还本续贷（新增，261版）

依据金规〔2024〕13号：在原贷款到期前签订新合同，以新发放贷款结清已有贷款。**不包括**贷款到期后的重组、展期。

## 规则线索

- **三表联动总量锚**：S63_I[1] 是 S64/S65 总量的源头，S64/S65 各行之和不能超过 S63_I[1]；调试三表差异时优先从 S63 入手
- **月报 vs 季报**：S63 第I部分是三表中唯一月报，频率最高，是监管对企业贷款余额变化的高频监测层
- **买断式转贴现的双层处理**：S63 中买断式转贴现既被排除在项目1-13之外，又作为独立项目[29]单独统计，说明监管需要同时知道"贷款含不含它"和"它自己有多大"两个维度
- **停产企业特殊处理**：S63 有"停产清算中企业参考停产前数据划型"的条款，S64 无；Query 涉及关停企业贷款时应优先查 S63
- **G01_II 作为上限约束**：S63 的企业贷款各项均受 G01_II 五级分类约束，说明 S63 不能"超出"资产负债表的贷款质量结构
