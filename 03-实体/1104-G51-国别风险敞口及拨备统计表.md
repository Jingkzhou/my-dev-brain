# 1104-G51-国别风险敞口及拨备统计表

## 简介

G51《国别风险敞口及拨备统计表》是 1104 监管报表体系中专门统计**国别风险债权、债务敞口及相关拨备**的报表，依据《银行业金融机构国别风险管理办法》制定。报表按国家/地区分列，覆盖直接债权→风险转移→最终债权→拨备的完整风险管理链条，并包含境外机构经营情况（人员、资产、利润等）和一带一路专项统计。

## 当前结论

- **并表口径，季后40日**：G51 仅要求并表口径（无法人口径），截止日与 G44 合并口径、G52 相同；此外，重大事件可随时报送，是 1104 体系中少数有"事件驱动报送"机制的报表
- **列结构 = 国别矩阵**：列 A 为全球合计；B-H 为七大国家组合（G7/金砖/上合/一带一路/RCEP/欧盟/OECD）；I 至 HI 为各具体国家/地区列（约200列）
- **三层债权分析框架**：直接境外债权 [1.] → 风险转移 [2.] → 最终境外债权 [3.] = [1.]+[2.]；风险转移净额可正可负
- **三维分解**：跨境债权（1.1）和境外分支机构债权（1.2）均可从债务人类型（6类）/ 融资工具类型（7类）/ 剩余期限（4档）三个维度展开
- **境外主体判定三原则**：成立地法律 / 实际管理机构地 / 主要经营活动地——主要以经营活动地为最终依据，离岸中心注册但经营在他处的按经营地
- **拨备计提五级体系**：低(0%) / 较低(0%) / 中等(≥5%) / 较高(≥15%) / 高(≥40%)；国家分类与G4A系列相互独立
- **附注内保外贷**：境内担保人对境外债务人的跨境担保，分融资性/非融资性/其他三类，含期末余额、签约额、履约额、垫款余额四列
- **填报机构范围较窄**：政策性银行、大型商业银行、股份制商业银行、城市商业银行、外资法人银行、外国银行分行——不含农村银行类、消费金融公司等

## 重点规则

### 境外主体国别判定规则

```
一般原则（按优先级）：
1. 主要经营活动所在国（最终判定依据）
2. 实际管理机构所在国（辅助）
3. 依据成立法律的国家（初步判断）

离岸中心注册但经营在他处 → 填列主要经营活动所在国
一国法律成立但经营在他国 → 填列经营所在国

分行处理：
- 境外企业的境内分公司（不能独立承担法律责任）→ 视为境外主体 → 债权填总公司所在国
- 中资/外资银行境外分行 → 直接债权填分行所在国，风险转移后最终债权归母行所在国
- 外资银行境内分行 → 直接填母行所在国，不作风险转移
```

### 三层债权框架

```
直接境外债权 [1.] = 跨境债权 [1.1] + 境外分支机构债权 [1.2]

风险转移 [2.] = 净转入 - 净转出 （可正可负）
  [2.1 跨境债权风险转移] = 2.1.1（转入）- 2.1.2（转出）
  [2.2 境外分支机构债权风险转移] = 2.2.1（转入）- 2.2.2（转出）

最终境外债权 [3.] = [1.] + [2.]
  [3.1 跨境债权] = [1.1] + [2.1]
  [3.2 境外分支机构债权] = [1.2] + [2.2]

特殊规则：由中国大陆主体提供担保的风险转移
  → 全额覆盖：最终债权中不再反映该笔债权
  → 部分覆盖：覆盖部分不再反映，未覆盖部分仍在直接债务人所在国体现
```

### 直接债权三维分解矩阵

| 维度 | 分类 |
|------|------|
| 债务人类型 | 银行 / 公共部门 / 非银行金融机构 / 非金融机构 / 个人 / 其他 |
| 融资工具 | 境外贷款（含银团）/ 存放+拆放+买入返售 / 存放中央银行 / 境外有价证券和基金 / 境外金融衍生品 / 对非并表境外机构投资 / 其他 |
| 剩余期限 | 1年以下（含） / 1-2年（含） / 2年以上 / 其他（无固定到期日） |

三个维度各自独立等于同一债权总量（类似G44的E列三套展开）。

### 拨备计提五级体系

| 国家分类 | 代码 | 最低拨备比率（正文）| 核对公式比率 |
|---------|------|-----------------|------------|
| 低 | 1 | 0% | 0.5%⚠️ |

计提公式：`[13.] = [12.] × [9.]`（应计提国别风险拨备 = 拨备比率 × 应计提拨备的国别风险资产）

满足《办法》第三十一条规定的可不计提，填0。

### 内保外贷附注

```
内保外贷 = 担保人注册地在境内 + 债务人和债权人注册地均在境外（汇发〔2014〕29号）

三类：
1. 融资性保函：借款保函、融资租赁保函、备用信用证（融资性）、保兑信用证等
2. 非融资性保函：投标、履约、预付款、海事、质量、关税、工程维修、诉讼、提货担保保函
3. 其他

四列：A=期末余额 / B=签约额（年累计） / C=履约额（年累计） / D=垫款余额（期末）
附注[4.] = [1.]+[2.]+[3.]
```

## 规则线索

- **G51 是1104体系中结构最复杂的报表之一**：31个主行项目 × 200+国家列 × 3个分解维度，加上附注内保外贷，数据量巨大；但核心逻辑是"直接债权→风险转移→最终债权→拨备"的四层递进
- **风险转移净额机制是国别风险的核心创新**：允许同一债权在不同国家列之间"流动"，对同一债务人同时计入"转出国（负数）"和"转入国（正数）"，实现风险归属的重新分配；这与信用风险的抵押品减值逻辑完全不同
- **中国大陆主体担保的特殊处理**：由大陆主体担保覆盖的境外债权，在最终债权中不再反映（等于该部分风险视为已由境内主体吸收），体现了监管对跨境担保国别风险转移的审慎认定
- **G51[6.A]=G40[3.A] 揭示 G40 报表的存在**：G40 为资本净额来源报表，目前尚未入库，是 G51 的强依赖项
- **重大事件随时报送机制**：G51 是1104体系中极少数具有"非定期报送触发条件"的报表，说明监管对跨境风险的实时关注度高于一般业务板块
- **外国银行分行的特殊口径**：[6.A] 资本净额外国银行分行按本外币合计口径（不同于法人银行的并表口径），且表间核对 G51[6.A]=G40[3.A] 不适用于外国银行分行，是易错点
- **拨备比率公式疑似版本差异**：正文规定（0%/0%/5%/15%/40%）与表内核对公式（0.5%/1%/15%/25%/50%）不一致，应以官方最新有效版本为准
