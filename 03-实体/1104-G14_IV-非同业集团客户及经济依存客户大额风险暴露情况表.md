# 1104-G14_IV-非同业集团客户及经济依存客户大额风险暴露情况表

## 简介

G14_IV《非同业集团客户及经济依存客户大额风险暴露情况表》是 1104 系统中的大额风险暴露专项报表部分，核心功能是反映非同业集团客户及经济依存客户中的大额风险暴露客户明细，补充了同业客户维度之外的风险暴露结构。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银保监统号（待确认）
- 报表名称：大额风险暴露统计表第 IV 部分：非同业集团客户及经济依存客户大额风险暴露情况表
- 整理日期：2026-04-11
- 来源状态：填报说明原文已入库

## 当前结论

- G14_IV 反映非同业集团客户及经济依存客户中大额风险暴露客户的明细情况，客户入表门槛为考虑合格风险缓释作用后风险暴露超过一级资本净额 2.5%
- 客户数量限制：≤70 家时全部填报；>70 家时仅填报前 70 家
- 排列顺序：按 `[D]考虑风险缓释作用的风险暴露总和` 降序排列
- 同一单一法人若同时属于非同业集团客户和经济依存客户，应按两类客户分别填报，可能导致同一客户出现两行
- 核心比例指标（E 列）以一级资本净额为分母，与 G40 存在表间勾稽：`[71.D] = G40_[2.A]`

| `[71.D]` | 一级资本净额 |

## 重点规则

### 表内校验关系

```text
[0] = [1] + [2] + ... + [70]       （汇总行等于明细行求和）
[D] = [F] + [G] + [H] + [I] + [J] + [K]  （各类风险暴露求和）
[G] = [G1] + [G2]                  （特定风险暴露分解）
[M] = [D] + [L]                    （风险缓释调整）
[E] = [D] / [71.D]                 （比例指标计算）
[E1] = [D1] / [71.D]               （不可豁免比例计算）
[1.D] ≥ [2.D] ≥ [3.D] ≥ ... ≥ [70.D]  （降序排列）
```

### 表间校验关系

```text
[71.D] = G40_[2.A]     （一级资本净额与 G40 对应行勾稽）
```

## 规则线索

- **大额风险暴露门槛管理**：2.5% 一级资本净额作为统一门槛，需与 G14_V（同业前100）、G14_VI（同业前30）进行对比
- **非同业 vs 同业分异**：G14_III（非同业单一）→ G14_IV（非同业集团+依存）的逻辑分解，形成对非同业客户的双维度覆盖
- **客户重复计算**：同一法人同时属于集团和依存时的分别填报规则，与风险敞口确认的关系待梳理
- **风险缓释应用**：合格风险缓释作用的定义及豁免对象细节需对应原办法条款
