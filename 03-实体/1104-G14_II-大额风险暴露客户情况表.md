# 1104-G14_II-大额风险暴露客户情况表

## 简介

G14_II《大额风险暴露客户情况表》是 G14 系列的"全量汇总视角"，反映银行所有类型的大额风险暴露客户明细，是 G14_III/IV/V/VI 各部分的去重补全版本，并额外纳入不考虑风险缓释作用口径下的客户。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银保监统号（待确认）
- 报表名称：大额风险暴露统计表第 II 部分：大额风险暴露客户情况表
- 整理日期：2026-04-11
- 来源状态：填报说明原文已入库

## 当前结论

- G14_II 是 G14 全量客户视角，覆盖所有类型：非同业单一、非同业集团、经济依存、同业单一、同业集团、资产管理产品、资产证券化产品、匿名客户
- 客户数量不定长，最低不少于 20 户（若大额风险暴露客户≤20个，填报前20大；若>20个，填报所有）
- **不是简单汇总 G14_III/IV/V/VI**：各部分存在"定长截断导致漏报"和"集团+单一重复计算"两类问题，本表单独设计，集团下单一客户不在本表单独填报，以准确统计全部大额风险暴露
- 纳入范围为：考虑风险缓释时超过 2.5% 的客户 **∪** 不考虑风险缓释时超过 2.5% 的客户（取并集）
- 对于仅因不考虑风险缓释才超阈值的客户，只需填 [H]、[I] 列；对于仅因考虑风险缓释才超阈值的客户，需填 [D]-[F6] 列
- 企业集团财务公司不需填报本部分
- 表间勾稽：[0.J] = G40_[2.A]（一级资本净额）

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 法人汇总数据为季报（季后18日内）；合并报表数据为半年报（半年后40日内） |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元、百分比（保留两位小数） |

### 适用机构

| 类别 | 机构范围 |
|------|----------|
| 适用机构 | 政策性银行、大型商业银行、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行、金融租赁公司、汽车金融公司、消费金融公司、农村资金互助社、贷款公司 |
| 不填报本部分机构 | **企业集团财务公司** |

### 客户筛选规则

| 项目 | 说明 |
|------|------|
| 纳入范围 | 考虑风险缓释时超过一级资本净额 2.5% **或** 不考虑风险缓释时超过一级资本净额 2.5% 的客户（取并集） |
| 数量规则 | 大额客户 ≤20 时填前20大；>20 时填报所有（最低不少于20户，不定长） |
| 排序要求 | 按 [D] 列降序排列 |
| 去重规则 | 集团客户下的单一客户不单独填报（避免重复计算） |

### 核心字段关系

| 字段 | 说明 |
|------|------|
| `[D]` | 考虑风险缓释作用的风险暴露总和 |
| `[D1]` | 不可豁免风险暴露（剔除办法第十三至十五条豁免部分，数值在 0～[D] 之间） |
| `[E]` | [D] / 一级资本净额 |
| `[E1]` | [D1] / 一级资本净额 |
| `[F1]～[F6]` | 一般、特定、交易账簿、交易对手信用、潜在、其他风险暴露 |
| `[G]` | 风险缓释转出的风险暴露（转入为负数，轧差填报） |
| `[H]` | 不考虑风险缓释的风险暴露总和 = [D] + [G] |
| `[I]` | [H] / 一级资本净额 |
| `[J]` | 一级资本净额（所有行相同） |

## 重点规则

### 表内校验关系

```text
[0] = [1]+[2]+...+[n]    （汇总行，适用于D、D1、F1-Q 各列）
[D] = [F1]+[F2]+[F3]+[F4]+[F5]+[F6]
[H] = [D] + [G]          （不考虑风险缓释 = 考虑风险缓释 + 缓释转出）
[E] = [D] / [J]
[E1] = [D1] / [J]
[I] = [H] / [J]
[1.D] ≥ [2.D] ≥ ... ≥ [n.D]   （降序排列）
[1.J] = [2.J] = ... = [n.J]    （一级资本净额各行相同）
```

### 表间校验关系

```text
[0.J] = G40_[2.A]    （一级资本净额与 G40 对应行勾稽）
```

## 规则线索

- **G14_II vs G14_III/IV/V/VI 的逻辑层次**：本表是全量且去重的汇总口径，其他各部分是按客户类型的明细口径，两者互补而非简单加总
- **不可豁免风险暴露 [D1]**：由《商业银行大额风险暴露管理办法》第十三至十五条定义豁免范围，待后续建立专项规则线索
- **客户代码规范**：境内法人填统一社会信用代码，自然人填身份证号，资产管理产品/资产证券化产品/匿名客户不填代码——B/C 字段说明具有跨部分通用性，其他部分（III/IV/V/VI）会引用本部分的 B/C 填报说明
