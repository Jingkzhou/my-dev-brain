# 1104-G33_Ⅱ-银行账簿利率风险计量报表（标准化计量框架完整版）

## 简介

G33_Ⅱ 是 1104 监管报表体系中的**银行账簿利率风险（IRRBB）**计量报表，依据原银保监会《商业银行银行账簿利率风险管理指引（修订）》设计。基于利率敏感性资产/负债的重定价现金流，计算6种利率冲击情景下的**经济价值变动**和**净利息收入变动**，适用于大型银行和股份制银行。

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：G33_Ⅱ

- 来源状态：填报说明原文已入库

- 适用机构：大型商业银行（含邮储银行）、股份制商业银行（非全口径报表）
- 数据单位：万元，金额保留两位小数

## 当前结论

- **限定机构范围**：仅大型行+股份制行，不含城商行、农商行等，是1104体系中少数有机构类型限制的报表
- **多货币分表机制**：主要货币（单一货币占银行账簿资产/负债比例>5%）需各自填报一张报表，折算为人民币；典型情景为人民币+美元+欧元三张
- **两类计量产出**：经济价值变动（[9.]，6种情景，取最大减少值作为监管关注指标）+ 净利息收入变动（[10.]，仅±250bps两种情景，人民币下移场景为"存款利率不变"的不对称情景）
- **行为性期权 vs 自动利率期权**：行为性期权（客户提前还款/提前支取的行为选择）需建模后调整现金流（填[1.2]/[1.3]/[2.3]/[2.4]）；自动利率期权（合同约定或嵌入式）需剥离并独立计量（填[11.]+[7.]）
- **无到期日存款建模**：须用10年历史数据区分核心存款（稳定且不随利率变化重定价）/非核心存款；非核心及非稳定部分划入【隔夜】；核心存款比例和期限有监管上限约束
- **衍生品复式记录**：所有衍生品（除利率期权外）拆为多头+空头分别填报；利率掉期典型案例：收浮付固 = 持有期限至最近重定价日的浮动多头 + 合约剩余期限的固定空头
- **自动生成项目**：[6.][8.][9.][10.] 均无需填报，系统自动计算

## 重点规则

### 主要货币判断规则

> 以该币种计价的银行账簿资产（或负债）余额占全部银行账簿资产（或负债）比重**超过5%**即为主要货币，需单独填报一张报表。

- 判断基准：资产端或负债端任一超过5%即触发
- 填报时均需折算为人民币，但在[0.]币种栏标注原币种
- 折算汇率：四大货币（USD/EUR/JPY/HKD）用外汇局基准汇价；其他货币用USD基准×上午9时国际市场套算

### 重定价日期确定规则

| 产品类型 | 本金重定价日 | 利息重定价日 |
|---------|------------|------------|
| 固定利率 | 合同到期日 | 每个利息支付日 |
| 浮动利率 | 距报告日最近的利率调整日 | 所有未偿还/未重定价本金的利息支付日 |
| 利率调整日不确定 | 紧接报告日的工作日 | 同上 |
| 逾期资产（非非应计） | 填入【隔夜-1个月】区间 | — |

**特殊情形**：浮动利率业务若已超过最后利率调整日，则以到期日为本金重定价日。

**利率上下限触发时**：浮动利率已触及上限或下限的，视同固定利率；重新浮动后再按浮动利率处理。

### 嵌入式利率期权剥离

```
含嵌入式期权的金融工具
  └── 剥离后的基础工具（本金+非期权现金流）→ 填 [1.] 或 [2.]
  └── 剥离出的嵌入式利率期权 → 填 [11.] 和 [7.]

示例：5年期固定利率可赎回债券（第4年末可按面值赎回）
  → 5年期固定利率债券（本金重定价期限5年）→ [1.1.2]
  → 4年期欧式看涨债券期权（执行价格=面值）→ [11.] + [7.]
```

### 行为性期权的贷款/存款处理

- **提前还款权贷款**：按零售/批发分类，建模计算6个利率冲击情景下的提前还款率，调整名义重定价现金流后填[1.2]/[1.3]
- **提前支取权存款**：按零售/批发分类，建模计算6个利率冲击情景下的提前支取率，调整现金流后填[2.3]/[2.4]
- 零售类定义：自然人贷款/存款 + 单位并表口径≤800万元的贷款/存款

### 无到期日存款建模

```
无到期日存款（活期）
  └── 非稳定存款 → 【隔夜】区间
  └── 稳定存款
        └── 核心存款（不随利率变化重定价）→ 根据历史数据确定到期期限，有监管上限
        └── 非核心存款（稳定但会随利率变化） → 【隔夜】区间
```

- 须使用10年历史数据（不足则用最长可获数据）
- 核心存款比例和平均期限须满足《银行账簿利率风险管理指引（修订）》标准化框架上限

### 衍生品复式记录典型案例

| 产品 | 多头填法 | 空头填法 |
|------|---------|---------|
| 利率掉期（收浮付固） | 期限=最近重定价日的浮动利率工具 | 期限=合约剩余期限的固定利率工具 |
| 远期外汇（5个月后卖出人民币买入美元） | 美元报表[3.1.1]（5个月） | 人民币报表[3.1.2]（5个月） |
| 债券期货多头 | 可交割债券（多头） | 期货交割日到期的零息债券（空头） |
| 6月份的3个月期利率期货多头（在4月） | 5个月期多头 | 2个月期空头 |

**非利率期权（如货币期权）**：使用delta等值（合约名义本金×delta）进行复式记录；债务工具期权用债务工具市值×delta。

### 利率期权经济价值计算

```
期权价值变动 = 基准利率情景下期权价值 - 特定冲击情景下期权价值

特定情景下全部自动利率期权价值变动
  = Σ（卖出自动利率期权价值变动）- Σ（买入自动利率期权价值变动）

注：计算时假设利率波动率 = 当期波动率 × 1.25
```

### 净利息收入变动的非对称情景

[10.净利息收入变动] 使用两种情景（非6种）：
1. **利率平行上移250个基点**：计算一年以内NII影响
2. **存款利率不变，其他利率平行下移250个基点**：人民币专项情景，体现存款利率不随市场利率对称下行的现实（利率管制特征）

## 规则线索

- **主要货币>5%判断**：银行外币业务占比报告中有时会用到类似概念，是否值得建独立规则页待观察（目前仅G33_Ⅱ涉及）
- **行为性期权建模**：提前还款/提前支取的建模方法是利率风险管理的核心难点，6种利率情景下分别测算提前行为率，体现了IRR管理与资本管理的差异；可与S67（房地产贷款）、G01（固定利率贷款存量）联动分析
- **经济价值 vs 净利息收入**：G33_Ⅱ 同时计量两类指标——EVE关注长期资本价值，NII关注短期盈利；这与巴塞尔委员会IRRBB标准（BCBS368）的双支柱框架一致
- **人民币NII下移情景的不对称设计**：存款利率不随市场利率等幅下降（利率管制/粘性），监管专门设计"存款利率不变+其他下移250bps"情景，是中国利率市场化不完全的监管表达
- **嵌入式期权剥离与G4C系列的衔接**：[3.]的衍生品填法参照G4C-1(b)（市场风险简化标准法），说明G33_Ⅱ在衍生品分类逻辑上与市场风险资本报表保持一致
