# 1104-G53-分地区情况表

## 简介

G53《分地区情况表》是 1104 监管报表体系中的地区维度综合报表，将多个核心业务视角（机构、利润、贷款行业、五级分类、融资担保、县域）按**省市自治区**维度展开，形成全国地理分布画像。报表文号：银监统0024号，分为六个部分，频率不同。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：G53（含 G53_I~G53_VI 六部分）
- 报表名称：分地区情况表
- 整理日期：2026-04-13
- 来源状态：填报说明原文已入库
- 报表文号：银监统0024号

## 当前结论

- **六部分结构，频率各异**：第一至四部分为季报（季后13日）；第五部分为半年报（半年后18日）；第六部分为季报但晚5天（季后18日）
- **机构范围有差异**：金融租赁公司仅填第三、四、六部分；货币经纪公司仅填第一、二部分
- **列结构统一**：按省市自治区（含计划单列市）分列；[AG]=境内分支机构合计，[AH]=境外分支机构合计（第六部分无境外列）
- **第三部分含四大区域专栏**：京津冀、长江经济带、粤港澳大湾区、自贸区（21个），是监管层跟踪重点区域信贷投放的关键数据源
- **第六部分县域定义**：依据2020年县域名单，含已脱贫人口/边缘易致贫人口/已脱贫地区/重点帮扶县四类精准扶贫口径
- **分支机构地区归属**：按设立注册地所在省份，不按业务发生地

### 第三部分四大区域专栏

第三部分在标准省份列之外，设有四个区域专栏：

- **京津冀地区**：北京+天津+河北
- **长江经济带**：沿长江11省市
- **粤港澳大湾区**：广东省内（珠三角）+ 港澳
- **自贸区**（21个）：上海、广东、天津、福建、辽宁、浙江、河南、湖北、重庆、四川、陕西、海南、山东、江苏、广西、河北、云南、黑龙江、北京、湖南、安徽

### 第六部分县域口径

```
县域 = 县 + 县级市 + 市辖区中的农村部分（依据2020年县域名单）

细分维度：
- 已脱贫人口贷款（含后续跟踪期）
- 边缘易致贫人口贷款
- 已脱贫地区贷款
- 重点帮扶县贷款
```

## 规则线索

- **G53 是地区维度的汇聚点**：第一至四部分分别与 G01/G04/G11_I/G11 口径一致，G53 的作用是将这些报表的数据拆解到省份粒度，形成地区分布视图；跨报表对比时，G53 的省份合计（AG+AH）应与对应基础报表总量相等
- **自贸区21个专栏是政策落地追踪**：自贸区数量随政策扩容而增加，填报时须关注最新自贸区名单；21个自贸区作为独立专栏，说明监管层对其有独立的信贷投放监控需求
- **县域扶贫口径的精准性**：第六部分含四类精准扶贫相关口径，说明 G53 县域部分不仅统计一般县域存贷款，还追踪政策目标群体，是普惠金融政策执行监测的数据基础
- **第五部分半年报**：融资担保公司担保贷款单独用半年报频率，且截止日为半年后18日（比一般季报的季后13日晚），暗示该数据需更多时间核实，也体现监管对融资担保代偿风险的持续关注
- **机构地区归属按注册地**：填报时分支机构按注册地归入省份，不按业务发生地，与 S26 省外融资"按最终使用地"的逻辑相反，是常见口径混淆点
