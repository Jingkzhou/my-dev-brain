# 1104-G01_II-贷款质量五级分类情况简表

## 简介

G01_II《贷款质量五级分类情况简表》是 1104 系统中的贷款质量专项报表，核心功能是反映报告期末各项贷款的五级分类和逾期贷款情况。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0002号
- 报表名称：资产负债项目表附注Ⅱ部分：贷款质量五级分类情况简表
- 整理日期：2026-04-08
- 来源状态：填报说明原文已入库

## 当前结论

- G01_II 按本外币合计人民币填报，反映贷款质量五级分类和逾期贷款情况。
- 核心拆分逻辑为“各项贷款 = 正常贷款 + 不良贷款”，其中正常贷款=正常类+关注类，不良贷款=次级类+可疑类+损失类。
- 逾期贷款部分不仅看逾期余额，还进一步拆分逾期60天以上和逾期90天以上贷款。
- 本表与 G01 主表存在直接勾稽关系：`[1.A] = G01_[62.C]`。

### 贷款五级分类

| 编号 | 项目 | 说明 |
|------|------|------|
| [1] | 各项贷款 | 包括贷款、贸易融资、票据融资、融资租赁、从非金融机构买入返售资产、透支、各项垫款等 |
| [1.1] | 正常贷款 | 正常类 + 关注类 |
| [1.1.1] | 正常类 | 债务人能够履行合同，没有客观证据表明本金、利息或收益不能按时足额偿付 |
| [1.1.2] | 关注类 | 存在不利影响因素，但债务人目前有能力偿付本金、利息或收益 |
| [1.2] | 不良贷款 | 次级类 + 可疑类 + 损失类 |
| [1.2.1] | 次级类 | 无法足额偿付本金、利息或收益，或金融资产已经发生信用减值 |
| [1.2.2] | 可疑类 | 已无法足额偿付本金、利息或收益，金融资产已发生显著信用减值 |
| [1.2.3] | 损失类 | 采取所有可能措施后，只能收回极少部分或损失全部金融资产 |

### 逾期贷款口径

| 编号 | 项目 | 说明 |
|------|------|------|
| [2] | 逾期贷款 | 报告期末未按贷款合同规定偿还本金或利息的各项贷款本金余额 |
| [2.1] | 逾期60天以上贷款 | 本金或利息逾期时间超过60天的各项贷款本金余额 |
| [2.2] | 逾期90天以上贷款 | 本金或利息逾期时间超过90天的各项贷款本金余额 |

### 关键口径说明

| 项目 | 说明 |
|------|------|
| 五级分类依据 | 以《贷款风险分类指引》《商业银行金融资产风险分类办法》相关要求为标准 |
| 个人消费贷款逾期 | 按月分期还款的个人消费贷款，逾期90天以内按已逾期部分本金填报，逾期91天及以上按整笔贷款本金填报 |
| 借新还旧逾期起算 | 借新还旧和因借新还旧而展期等形式的贷款，逾期天数从原借款合同到期日起算 |
| 正常展期逾期起算 | 正常贷款展期的逾期天数从展期后到期日起算 |
| 逾期判断 | 若一笔贷款本金和利息都逾期，按逾期时间较长的一端判断 |

## 重点规则

### 表内校验关系

```text
[1.] = [1.1] + [1.2]    （各项贷款 = 正常贷款 + 不良贷款）
[1.1] = [1.1.1] + [1.1.2]    （正常贷款 = 正常类 + 关注类）
[1.2] = [1.2.1] + [1.2.2] + [1.2.3]    （不良贷款 = 次级类 + 可疑类 + 损失类）
```

